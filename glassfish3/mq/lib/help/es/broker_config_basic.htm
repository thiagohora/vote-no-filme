Básicas

Use esta ficha para especificar o examinar la información de configuración básica de una instancia de agente.

Versión Especifica la versión del agente.

Nombre de instancia Especifica el nombre predeterminado imqbroker o del nombre que asignara al iniciar el agente.

Puerto primario Especifica el número del puerto del asignador de puertos del agente. El asignador de puertos se ocupa de asignar puertos dinámicamente a los servicios Message Queue. Debe ser el mismo número que indicó al iniciar el agente.

Crear temas automáticamente Especifica si el agente puede crear un destino de tema cada vez que un consumidor o productor de mensajes intente obtener acceso a un destino inexistente.

Crear colas automáticamente Especifica si el agente puede crear un destino de cola cada vez que un consumidor o productor de mensajes intente obtener acceso a un destino inexistente.

Número máximo de consumidores activos de la cola creada automáticamente Especifica el número máximo de consumidores que puede estar activo en una entrega de carga equilibrada desde un destino de cola.

Número máximo de consumidores de respaldo de la cola creada automáticamente Especifica el número máximo de consumidores de respaldo que puede ocupar el lugar de los consumidores activos en caso de que uno de estos últimos falle durante una entrega de carga equilibrada desde este destino.
