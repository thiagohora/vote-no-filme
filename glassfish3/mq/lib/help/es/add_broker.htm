Agregar agente

Use este diálogo para crear una representación de un agente en el panel de navegación de la consola de administración. No es necesario iniciar el agente para ello. Deberá iniciarlo si desea conectarse a él.

Etiqueta de agente Especifica el nombre que desea que lleve este agente en el panel de navegación de la consola de administración. Este nombre no tiene por qué ser el mismo que el nombre de instancia del agente.

Host Especifica el nombre del equipo que aloja este agente. También puede mostrar el valor predeterminado localhost.

Puerto primario Especifica el número del puerto asignado al asignador de puertos. De forma predeterminada, éste es 7676. La combinación de nombre de host y número de puerto primario identifica de manera única al agente. Para poder ejecutar dos instancias de un agente en un mismo servidor, el asignador de puertos de cada una de ellas debe tener distinto número de puerto.

Nombre de usuario Especifica su nombre de usuario. De forma predeterminada, éste es admin para el administrador. Esta información se usa para autenticarle cuando intente conectar con el agente. Este paso no es seguro. Si le preocupa la seguridad, debe esperar hasta el momento de la conexión, en que se le pedirá esta información.

Contraseña Especifica su contraseña. De forma predeterminadaadmin para el administrador. Esta información se usa para autenticarle cuando intente conectar con el agente. Este paso no es seguro. Si le preocupa la seguridad, debe esperar hasta el momento de la conexión, en que se le pedirá esta información.

  • Use el botón Aceptar para confirmar la información introducida y agregar el agente.

Para conectar con el agente agregado:

  1. Seleccione el agente en el panel de navegación de la consola de administración.
  2. Elija Acciones > Conectar con agente.

Para consultar y/o actualizar un agente en ejecución:

  1. Seleccione el agente en el panel de navegación de la consola de administración.
  2. Conéctese a él si aún no lo ha hecho.
  3. Elija Acciones > Consultar/Actualizar agente.

Para cambiar la información indicada en este cuadro de diálogo para un agente particular, haga lo siguiente:

  1. Seleccione el agente en el panel de navegación.
  2. Desconéctese del agente si aún no lo ha hecho. Un agente está desconectado si su icono en el panel de navegación tiene este aspecto:
    Para desconectar un agente, elija Acciones > Desconectar de agente
  3. Elija Acciones > Propiedades para abrir el cuadro de diálogo Propiedades del agente.
