Propiedades del servicio

Use este diálogo para especificar las propiedades del servicio Message Queue seleccionado. Puede cambiar cualquiera de las propiedades siguientes sin detener en pausa el servicio.

Número de puerto Especifica el número de puerto que se asignó dinámicamente al servicio seleccionado por el asignador de puertos. Puede hacer clic en el botón de radio Estático e indicar un número de puerto alternativo (disponible) para el servicio.

Estado del servicio Indica el estado actual del servicio.

Número actual de subprocesos asignados Especifica el número de subprocesos asignados a este servicio.

Número actual de conexiones Especifica el número de conexiones asignadas actualmente a este servicio.

Número mín. de subprocesos Especifique el número mínimo de subprocesos que deben asignarse del grupo de subprocesos para este servicio.

Número máx. de subprocesos Especifique el número máximo de subprocesos que deben asignarse del grupo de subprocesos para este servicio.

Haga clic en Aceptar para definir estos valores de configuración del servicio seleccionado.
