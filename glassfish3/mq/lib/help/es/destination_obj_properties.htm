Propiedades del objeto de destino

Use este diálogo para cambiar las propiedades de un objeto de destino.

Sólo lectura Haga clic en esta casilla de verificación para activar el estado de sólo lectura del objeto. Si un objeto está marcado como de sólo lectura, las aplicaciones cliente no podrán modificar los atributos de objetos administrados mediante llamadas de API.

Nombre del destino Use este campo para especificar el nombre del destino físico en el agente al que corresponde este objeto administrado. Si no lo especifica, se usará el nombre predeterminado de Untitled_Destination_Object.

Descripción del destino Use este campo para indicar una descripción útil de este objeto de destino.

Haga clic en el botón Aceptar para reconfigurar el objeto de destino usando los valores especificados en el diálogo.
