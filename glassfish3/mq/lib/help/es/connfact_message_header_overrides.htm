Propiedades del objeto de fábrica de conexión > Valores de reemplazo de los encabezados de mensaje

Use esta ficha para reemplazar los campos de encabezados de mensaje JMS que especifican la persistencia, duración y prioridad de los mensajes. Debido a que reemplazar estos valores puede interferir con los usuarios o diseñadores de aplicaciones actuales, coordínese con ellos antes de realizar estos cambios.

Anular JMSDeliveryMode Especifica si se puede reemplazar el valor definido por el cliente para JMSDeliveryMode. Por defecto: falso.

Valor de JMSDeliveryMode Especifica el valor de reemplazo para JMSDeliveryMode. Los valores son 1 (no persistente) y 2 (persistente). Por defecto: 2.

Anular JMSExpiration Especifica si se puede reemplazar el valor definido por el cliente para JMSExpiration. Por defecto: falso.

Valor de JMSExpiration Especifica el valor de reemplazo para JMSExpiration (en milisegundos). Por defecto: 0 (no vence).

Anular JMSPriority Especifica si se puede reemplazar el valor definido por el cliente para JMSPriority. Por defecto: falso.

Valor de JMSPriority Especifica el valor de reemplazo de JMSPriority (un entero de 0 a 9). Por defecto: 4 (normal).

Reemplazar mensajes a destinos temporales Especifica si los reemplazos se aplican a los destinos temporales. Por defecto: falso.
