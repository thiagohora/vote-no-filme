Propiedades del almacén de objetos

Use este diálogo para agregar, eliminar o actualizar las propiedades de un almacén de objetos que ya haya agregado al panel de navegación de la consola de administración. Para editar los campos del diálogo, desconecte la consola de administración del almacén de objetos y vuelva a abrir de nuevo el diálogo.

Propiedades del servicio de nombres JNDI: Use los controles de esta área para definir la lista de propiedades JNDI para este almacén de objetos. Use la lista desplegable Nombre para seleccionar un nombre de propiedad. Después de seleccionar el nombre de propiedad, use el campo Valor para escribir el valor de la propiedad.

Consulte http://java.sun.com/products/jndi/1.2/javadoc/ para obtener información acerca de la interfaz de programación de aplicaciones JNDI. Deberá definir las propiedades java.naming.factory.initial y java.naming.provider.url.

La propiedad de contexto inicial (java.naming.factory.initial) puede tener dos valores:

  • Almacén del sistema de archivos: com.sun.jndi.fscontext.RefFSContextFactory
  • Servidor LDAP: com.sun.jndi.ldap.LdapCtxFactory

También deberá especificar la información de autenticación. Sin embargo, si le preocupa la seguridad, puede esperar a que se le pida dicha información en el momento de la conexión.

  • Después de especificar el nombre de propiedad y su valor, haga clic en el botón Agregar para agregar la propiedad a la tabla.
  • Para eliminar una propiedad existente, selecciónela en la tabla y haga clic en el botón Eliminar para eliminarla.
  • Use el botón Borrar para borrar todas las entradas.
  • Seleccione la propiedad que desee, cambie su valor y use el botón Cambiar para reconfigurarla.
  • Use el botón Aceptar para confirmar todos los valores de las entradas.

Para conectar de nuevo con el almacén de objetos después de reconfigurarlo: Seleccione el almacén en el panel de navegación y elija Acciones > Conectar con almacén de objetos.
