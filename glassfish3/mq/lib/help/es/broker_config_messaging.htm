Mensajería

Use esta ficha para configurar la capacidad de agente y el tamaño de los mensajes para esta instancia del agente.

Número máx. de mensajes en el sistema Especifica el número máximo de mensajes en memoria y en disco (debido al intercambio). Se rechazará cualquier mensaje adicional. El valor predeterminado es "Ilimitado".

Tamaño máx. de mensajes en el sistema Especifica el tamaño total máximo (en bytes, kilobytes o megabytes) de mensajes en memoria y en disco (debido al intercambio). Se rechazará cualquier mensaje adicional. El valor predeterminado es "Ilimitado". Use la lista desplegable para indicar las unidades empleadas para calcular el tamaño: bytes, kilobytes o megabytes.

Tamaño máximo de mensaje Especifica el tamaño máximo permitido (en bytes, kilobytes o megabytes) para el cuerpo de un mensaje. Cualquier mensaje mayor que este límite será rechazado. El valor predeterminado es 70 megabytes. Use la lista desplegable para indicar las unidades empleadas para calcular el tamaño: bytes, kilobytes o megabytes.
