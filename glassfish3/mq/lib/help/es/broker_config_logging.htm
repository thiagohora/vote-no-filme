Registro

Use esta ficha para configurar el registro de una instancia de agente.

Nivel de registro Especifica el nivel de información de registro que se puede escribir en un canal de salida. Las categorías, por orden decreciente de gravedad, son ERROR, ADVERTENCIA, INFO. Utilice la lista desplegable para seleccionar una categoría. Al seleccionar una categoría, el generador de registros produce los datos de la categoría especificada y de todas las categorías superiores. Por ejemplo, si selecciona INFO, obtendrá datos en las tres categorías. Si selecciona ERROR, sólo obtendrá mensajes de este tipo. El valor predeterminado es INFO.

Tamaño de reinicio ('rollover') del registro Especifica el tamaño del archivo de registro a partir del cual se inicia un archivo de registro nuevo. Utilice la lista desplegable para seleccionar las unidades: bytes, kilobytes o megabytes. Seleccione el botón de radio "Ilimitado" para indicar que no se realice un cambio de archivo de registro basándose en su tamaño. La selección predeterminada es "Ilimitado".

Intervalo de reinicio ('rollover') del registro Especifica la duración, en segundos, del archivo de registro cuando se crea un nuevo registro. Seleccione el botón de radio "Ilimitado" para indicar que no se realice un cambio de archivo de registro basándose en su duración. El valor predeterminado es 604800.
