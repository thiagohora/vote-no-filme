Propiedades del agente

Use este diálogo para cambiar la representación de un agente en el panel de navegación de la consola de administración. No es necesario iniciar el agente para ello. Para editar los campos del diálogo, desconecte la consola del agente y vuelva a abrir de nuevo el diálogo.

Etiqueta de agente Especifica el nombre que desea que lleve este agente en el panel de navegación de la consola de administración. Este nombre no tiene por qué ser el mismo que el nombre de instancia del agente.

Host Especifica el nombre del equipo que aloja este agente. También puede mostrar el valor predeterminado localhost.

Puerto primario Especifica el número del puerto asignado al asignador de puertos. La combinación de nombre de host y número de puerto primario identifica de manera única al agente. Para poder ejecutar dos instancias de un agente en un mismo servidor, el asignador de puertos de cada una de ellas debe tener distinto número de puerto.

Nombre de usuario Especifica su nombre de usuario. De forma predeterminada, éste es admin para el administrador. Esta información se usa para autenticarle cuando intente conectar con el agente. Este paso no es seguro. Si le preocupa la seguridad, debe esperar hasta el momento de la conexión, en que se le pedirá esta información.

Contraseña Especifica su contraseña. De forma predeterminada, éste es admin para el administrador. Esta información se usa para autenticarle cuando intente conectar con el agente. Este paso no es seguro. Si le preocupa la seguridad, debe esperar hasta el momento de la conexión, en que se le pedirá esta información.

Para volver a conectar con el agente:

  1. Seleccione el agente en el panel de navegación de la consola de administración.
  2. Elija Acciones > Conectar con agente.
