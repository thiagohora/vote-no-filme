Agregar almacén de objetos

Use este diálogo para agregar un almacén de objetos existente al panel de navegación de su herramienta de administración. Después de agregar el almacén, puede conectarse a él y agregarle objetos de destino y objetos de fábrica de conexión.

Un almacén de objetos contiene los objetos administrativos que usted crea: objetos de destino y objetos de fábrica de conexión. Su contenido es accesible mediante código estándar de búsqueda JNDI ejecutado por la aplicación cliente. Un almacén de objetos puede estar basado en archivos o ser un servidor LDAP. Si desea utilizar un almacén basado en archivos, deberá crear una carpeta para el mismo. Si desea usar un servidor LDAP, tendrá que configurar e iniciar el servidor.

Etiqueta del almacén de objetos: Especifica el nombre del almacén de objetos. Este nombre aparecerá en el panel de navegación de la consola de administración.

Propiedades del servicio de nombres JNDI: Use los controles de esta área para definir la lista de propiedades JNDI para este almacén de objetos. Use la lista desplegable Nombre para seleccionar un nombre de propiedad. Después de seleccionar el nombre de propiedad, use el campo Valor para escribir el valor de la propiedad.

Consulte http://java.sun.com/products/jndi/1.2/javadoc/ para obtener información acerca de la interfaz de programación de aplicaciones JNDI. Deberá definir las propiedades java.naming.factory.initial y java.naming.provider.url.

La propiedad de contexto inicial (java.naming.factory.initial) puede tener dos valores:

  • Almacén del sistema de archivos: com.sun.jndi.fscontext.RefFSContextFactory
  • Servidor LDAP: com.sun.jndi.ldap.LdapCtxFactory

También deberá especificar la información de autenticación. Sin embargo, si le preocupa la seguridad, puede esperar a que se le pida dicha información en el momento de la conexión.

  • Después de especificar el nombre de propiedad y su valor, haga clic en el botón Agregar para agregar la propiedad a la tabla.
  • Para eliminar una propiedad existente, selecciónela en la tabla y haga clic en el botón Eliminar para eliminarla.
  • Use el botón Borrar para borrar todas las entradas.
  • Para reconfigurar una propiedad, seleccione la propiedad que desee, cambie sus valores y use el botón Cambiar para reconfigurarla.
  • Use el botón Aceptar para confirmar los valores de la propiedad y agregar el almacén de objetos.

Para cambiar las propiedades de un almacén de objetos existente, haga lo siguiente:

  1. Desconéctese del almacén de objetos si aún no lo ha hecho. Un almacén de objetos está desconectado si su icono en el panel de navegación tiene este aspecto:
    Para desconectarse de un almacén de objetos, seleccione el almacén en el panel de navegación y elija Acciones > Desconectar del almacén de objetos.
  2. Elija Acciones > Propiedades para abrir el diálogo Propiedades del almacén de objetos.

Para agregar un objeto de destino al almacén de objetos, haga clic en la carpeta Destinos y elija Acciones > Agregar objeto de destino.

Para agregar una fábrica de conexión al almacén de objetos, haga clic en la carpeta Fábrica de conexión y elija Acciones > Agregar objeto de fábrica de conexión.
