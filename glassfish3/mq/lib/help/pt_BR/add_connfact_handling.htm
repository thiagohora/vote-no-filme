Manipulação de conexões

Utilize a aba Manipulação de conexões para configurar as conexões que serão criadas por esta fábrica.

Lista de endereços do servidor de mensagens Especifica uma lista de endereços do servidor de mensagens (uma ou mais), separados por vírgulas, cada um correspondente a uma instância diferente do broker à qual o tempo de execução do cliente pode se conectar. Cada endereço da lista especifica o nome do host, a porta do host e o serviço de conexão da conexão.
Para obter mais informações sobre a Sintaxe de endereço do servidor de mensagens, clique aqui

Ordem da lista de endereços Especifica se as tentativas de conexão são realizadas na ordem em que aparecem os endereços na lista de endereços do servidor de mensagens (PRIORIDADE) ou em uma ordem aleatória (ALEATÓRIA). Se houver muitos clientes tentando se conectar utilizando a mesma fábrica de conexões, a ordem aleatória deveria ser utilizada para impedi-los de conectarem-se ao mesmo endereço.

Número de iterações da lista de endereços Especifica o número de vezes que o tempo de execução do cliente realizará iterações na lista de endereços do servidor de mensagens para tentar estabelecer ou restabelecer uma conexão. Um valor de -1 indica que o número de tentativas é ilimitado.

Ativar reconhecimento automático para servidor de mensagens Um valor definido como verdadeiro especifica que o tempo de execução do cliente tentará se reconectar a um servidor de mensagens (ou à lista de endereços em uma lista de servidores de mensagens) quando a conexão for perdida.

Número de tentativas de reconexão por endereço Especifica o número de tentativas de reconexão que será realizado para cada endereço da lista de endereços do servidor de mensagens antes que o tempo de execução do cliente passe para o próximo endereço da lista. Um valor de -1 indica que o número de tentativas é ilimitado.

Intervalo de reconexão por endereço (milissegundos) Especifica o intervalo (em milissegundos) entre as tentativas de reconexão para cada endereço da lista de endereços do servidor de mensagens antes que o tempo de execução do cliente passe para o próximo endereço da lista. Se for muito curto, esse intervalo de tempo não dará ao broker tempo para se recuperar. Se for muito longo, a reconexão pode significar um atraso inaceitável.

Intervalo da conexão com ping (em segundos) Especifica o intervalo mínimo (em segundos) no qual o tempo de execução do cliente verificará periodicamente se a sua conexão com o broker está funcionando.
