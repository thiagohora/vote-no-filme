Básico

Utilize esta aba para especificar ou explorar as informações de configuração básicas de uma instância do broker.

Versão Especifica a versão do broker.

Nome da instância Especifica o nome padrão imqbroker ou o nome que você atribuiu quando o broker for iniciado.

Porta primária Especifica o número da porta Port Mapper do broker. O Port Mapper é responsável por atribuir dinamicamente portas aos serviços do Message Queue. Deveria ser o mesmo número especificado quando o broker foi iniciado.

Criar tópicos automaticamente Especifica se o broker pode criar um destino de tópico sempre que um consumidor de mensagem ou um produtor de mensagem tentar acessar um destino inexistente.

Criar filas automaticamente Especifica se o broker pode criar um destino de fila sempre que um consumidor de mensagem ou um produtor de mensagem tentar acessar um destino inexistente.

Número máximo de consumidores ativos da fila criado automaticamente Especifica o número máximo de consumidores que podem estar ativos nas entregas com balanceamento de carga a partir de um destino de fila.

Número máximo de consumidores de backup da fila criado automaticamenteEspecifica o número máximo de consumidores de backup que podem tomar o lugar dos consumidores ativos, caso qualquer consumidor ativo falhe durante a entrega com balanceamento de carga a partir do destino de fila.
