Adicionar objeto de fábrica de conexões > Substituições do cabeçalho de mensagens

Utilize esta aba para substituir os campos de cabeçalho das mensagens JMS que especificam a persistência, o tempo de vida e a prioridade das mensagens. Como a substituição destes valores pode afetar os usuários atuais ou os designers do aplicativo, entre em acordo com eles antes de realizar estas alterações.

Ignorar JMSDeliveryMode Especifica se o campo JMSDeliveryMode definido pelo cliente pode ser substituído. Padrão: falso

Valor JMSDeliveryMode Especifica o valor de substituição do JMSDeliveryMode. Os valores são 1 (não persistente) ou 2 (persistente). Padrão: 2.

Ignorar JMSExpiration Especifica se o campo JMSExpiration definido pelo cliente pode ser substituído. Padrão: falso

Valor JMSExpiration Especifica o valor de substituição do JMSExpiration (em milissegundos). Padrão: 0 (não expira).

Ignorar JMSPriority Especifica se o campo JMSPriority definido pelo cliente pode ser substituído. Padrão: falso

Valor JMSPriority Especifica o valor de substituição do JMSPriority (um inteiro de 0 a 9). Padrão: 4 (normal).

Ignorar mensagens dos destinos temporários Especifica se a substituição se aplica aos destinos temporários. Padrão: falso
