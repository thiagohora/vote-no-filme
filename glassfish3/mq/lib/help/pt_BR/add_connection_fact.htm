Adicionar objeto de fábrica de conexões

Utilize esta caixa de diálogo para configurar um objeto de fábrica de conexões e para adicioná-lo a um armazenamento de objetos.

As fábricas de conexões são objetos administrados utilizados por clientes JMS para se conectarem ao broker. Este objeto também é utilizado para especificar como a conexão se comporta e como o tempo de execução do cliente que o utiliza acessa o broker.

Nome de pesquisa Utilize este campo para especificar o nome de pesquisa JNDI do objeto de fábrica de conexões.

Tipo de fábrica Escolha o item de menu que especifica o tipo de conexões que a fábrica produz: ConnectionFactory, QueueConnectionFactory ou TopicConnectionFactory. Se a mensagem deve fazer parte de uma transação distribuída, escolha XAConnectionFactory, XAQueueConnectionFactory ou XATopicConnectionFactory.

Somente leitura Clique nesta caixa de seleção para tornar somente leitura o objeto de fábrica de conexões. Se o objeto for somente leitura, os aplicativos clientes não poderão modificar os atributos da fábrica de conexões através de chamadas API.

Ao configurar um objeto de fábrica de conexões, você especifica as propriedades comuns de todas as conexões que a fábrica produz. Utilize as abas mostradas na caixa de diálogo para configurar a fábrica de conexões. Clique em um dos links seguintes para obter informações sobre uma determinada aba.

  • Clique no botão Restaurar para padrões para restaurar todos os campos das abas da caixa de diálogo com seus valores padrão.
  • Clique em OK para criar o objeto de fábrica de conexões.

Para alterar as propriedades de um objeto de fábrica de conexões:

  1. Selecione Fábricas de conexões no painel de navegação do console de administração.
  2. Selecione o objeto de fábrica de conexões no painel de resultados do console de administração.
  3. Escolha Ações > Propriedades para abrir a caixa de diálogo Propriedades do objeto de fábrica de conexões.
