Registro

Utilize esta aba para configurar o registro de uma instância do broker.

Nível de log Especifica o nível de saída de registro que pode ser gravado em qualquer canal de saída. As categorias, em ordem decrescente de gravidade, são ERRO, WARNING, INFO. Utilize a lista suspensa para selecionar uma categoria. Quando uma categoria for selecionada, o logger grava os dados da categoria especificada, bem como das categorias superiores. Por exemplo, se selecionar INFO, você obterá os dados das três categorias. Se selecionar ERRO, obterá somente as mensagens de ERRO . O padrão é INFO.

Tamanho de rolagem do log Especifica o tamanho do arquivo de log file no qual a saída passa para um novo arquivo de log. Utilize a lista suspensa para selecionar as unidades: bytes, kbytes ou megabytes. Selecione o botão de opção Ilimitado para não especificar nenhuma rolagem sobre o tamanho do arquivo de log. A seleção padrão é Ilimitado.

Intervalo de rolagem do log Especifica o tempo, em segundos, do arquivo de log quando a saída passa para um novo arquivo de log. Selecione o botão de opção Ilimitado para não especificar nenhuma rolagem sobre o tempo do arquivo de log. O padrão é 604800.
