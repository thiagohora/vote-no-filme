Propriedades do broker

Utilize esta caixa de diálogo para alterar a representação do broker no painel de navegação do console de administração. Não é necessário iniciar o broker para isso. Para editar os campos desta caixa de diálogo, é necessário desconectar o console de administração do broker e exibir esta caixa de diálogo novamente.

Rótulo do broker especifica o nome que você gostaria que fosse exibido para este no painel de navegação do console de administração. Este nome não precisa ser igual ao nome da instância do broker.

Host Especifica o nome da máquina que hospeda este broker ou deixe o valor padrão de localhost.

Porta primária Especifica o número da porta atribuída ao Port Mapper. A combinação do nome do host e do número da porta primária identifica exclusivamente o broker. Para executar duas instâncias de um broker no mesmo servidor, cada Port Mapper deve ter um número de porta diferente.

Nome de usuário Especifica o seu nome de usuário. Por padrão, o nome é admin para o administrador. Esta informação é utilizada para autenticá-lo quando se conectar ao broker. Introduzi-la nesta caixa de diálogo não é seguro. Se for uma pessoa preocupada com a segurança, espere até o momento da conexão quando você será solicitado a fornecer esta informação.

Senha Especifica a sua senha de usuário. Por padrão, o nome é admin para o administrador. Esta informação é utilizada para autenticá-lo quando se conectar ao broker. Introduzi-la nesta caixa de diálogo não é seguro. Se for uma pessoa preocupada com a segurança, espere até o momento da conexão quando você será solicitado a fornecer esta informação.

Para reconectar ao broker:

  1. Selecione o broker no painel de navegação do console de administração.
  2. Escolha Ações > Conectar ao broker.
