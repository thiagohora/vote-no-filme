Propriedades de objeto de destino

Utilize esta caixa de diálogo para alterar as propriedades de um objeto de destino.

Somente leitura Clique nesta caixa de seleção para alternar o status de somente leitura do objeto. Se o objeto estiver marcado como somente leitura, os aplicativos clientes não poderão modificar os atributos do objeto administrado através de chamadas API.

Nome do destino Utilize este campo para especificar o nome do destino físico no broker ao qual corresponde o objeto administrado. Se você não especificar um nome, o nome padrão Untitled_Destination_Object será utilizado.

Descrição do destino Utilize este campo para introduzir uma descrição útil deste objeto de destino.

Clique no botão OK para reconfigurar o objeto de destino usando os valores especificados na caixa de diálogo.
