Mensagens

Utilize esta aba para configurar a capacidade do sistema de mensagens e o tamanho das mensagens desta instância do broker.

Número máximo de mensagens no sistema Especifica o número máximo de mensagens na memória e no disco (devido à permuta). As mensagens adicionais serão rejeitadas. O padrão é Ilimitado.

Total máximo de bytes das mensagens no sistema Especifica o tamanho total máximo (em bytes, Kbytes ou Mbytes) das mensagens na memória e no disco (devido à permuta). As mensagens adicionais serão rejeitadas. O padrão é Ilimitado. Utilize a lista suspensa para especificar as unidades utilizadas para calcular o tamanho: bytes, kbytes ou megabytes.

Tamanho máximo da mensagem Especifica o tamanho máximo permitido (em bytes, Kbytes ou Mbytes) do corpo de uma mensagem. Qualquer mensagem maior do que o tamanho definido nesta opção será rejeitada. O padrão é 70 megabytes. Utilize a lista suspensa para especificar as unidades utilizadas para calcular o tamanho: bytes, kbytes ou megabytes.
