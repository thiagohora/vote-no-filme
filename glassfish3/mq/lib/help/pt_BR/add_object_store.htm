Adicionar armazenamento de objeto

Utilize esta caixa de diálogo para adicionar um armazenamento de objeto existente ao painel de navegação da sua ferramenta administrativa. Após adicionar o armazenamento de objeto, é possível conectá-lo e passar a adicionar a ele objetos de destino e fábrica de conexões.

Um armazenamento de objeto contém os objetos administrativos criados: objetos de destino e objetos de fábrica de conexões. Seu conteúdo é acessado através do código de pesquisa JNDI padrão chamado pelo aplicativo cliente. Um armazenamento de objeto pode ter um arquivo como base ou pode ser um servidor LDAP. Se quiser utilizar um armazenamento com base em arquivo, é necessário criar uma pasta para ele. Se quiser utilizar um servidor LDAP, é necessário configurar e iniciar tal servidor.

Rótulo do armazenamento de objeto: Especifica o nome do armazenamento de objeto. Esse nome será exibido no painel de navegação do console de administração.

Propriedades do serviço de nomeação JNDI: Utilize os controles deste área para definir a lista de propriedades JNDI do armazenamento de objetos. Utilize a lista suspensa Nome para selecionar um nome de propriedade. Após selecionar o nome da propriedade, utilize o campo Valor para inserir o valor da propriedade.

Consulte http://java.sun.com/products/jndi/1.2/javadoc/ para obter informações sobre a interface de programação de aplicativo JNDI. Você é solicitado a definir as propriedades java.naming.factory.initial e java.naming.provider.url .

A propriedade de contexto inicial (java.naming.factory.initial) pode ter dois valores:

  • Armazenamento do sistema de arquivos: com.sun.jndi.fscontext.RefFSContextFactory
  • Servidor LDAP: com.sun.jndi.ldap.LdapCtxFactory

Você também é solicitado a especificar informações de autenticação. No entanto, se estiver preocupado com a segurança, espere até ser solicitado a inserir estas informações no momento da conexão.

  • Após especificar o nome e o valor da propriedade, clique no botão Adicionar para adicionar a propriedade à tabela.
  • Para excluir uma propriedade que já foi adicionada, selecione uma propriedade na tabela e clique no botão Excluir.
  • Utilize o botão Limpar para limpar todas as entradas.
  • Para reconfigurar uma propriedade, selecione uma entrada, altere seu valor e utilize o botão Alterar para reconfigurar tal propriedade.
  • Utilize o botão OK para enviar todas as configurações da propriedade e adicionar o armazenamento de objetos.

Para alterar as propriedades de um armazenamento de objetos existente, você deve fazer o seguinte:

  1. Deconecte-se do armazenamento de objetos se ainda não estiver desconectado. O armazenamento de objetos está desconectado se, no painel de navegação, o ícone aparecer da seguinte forma:
    Para desconectar um armazenamento de objeto, selecione o armazenamento no painel de navegação e escolha Ações > Desconectar do armazenamento de objeto.
  2. Escolha Ações > Propriedades para exibir a caixa de diálogo Propriedades do armazenamento de objeto.

Para adicionar um objeto de destino ao armazenamento de objeto, clique na pasta Destinos e escolha Ações > Adicionar objeto de destino.

Para adicionar uma fábrica de conexões ao armazenamento de objeto, clique na pasta Fábrica de conexões Ações > Adicionar objeto de fábrica de conexões.
