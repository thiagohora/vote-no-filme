Adicionar broker

Utilize esta caixa de diálogo para criar uma representação de um broker no painel de navegação do console de administração. Não é necessário iniciar o broker para isso. Será necessário iniciá-lo antes de conectá-lo.

Rótulo do broker especifica o nome que você gostaria que fosse exibido para este no painel de navegação do console de administração. Este nome não precisa ser igual ao nome da instância do broker.

Host Especifica o nome da máquina que hospeda este broker ou deixa o valor padrão de localhost.

Porta primária Especifica o número da porta atribuída ao Port Mapper. Por padrão este número é 7676. A combinação do nome do host e do número da porta primária identifica exclusivamente o broker. Para executar duas instâncias de um broker no mesmo servidor, cada Port Mapper deve ter um número de porta diferente.

Nome de usuário Especifica o seu nome de usuário. Por padrão, o nome é admin para o administrador. Esta informação é utilizada para autenticá-lo quando se conectar ao broker. Introduzi-la nesta caixa de diálogo não é seguro. Se for uma pessoa preocupada com a segurança, espere até o momento da conexão quando você será solicitado a fornecer esta informação.

Senha Especifica a sua senha de usuário. Por padrão, o nome é admin para o administrador. Esta informação é utilizada para autenticá-lo quando se conectar ao broker. Introduzi-la nesta caixa de diálogo não é seguro. Se for uma pessoa preocupada com a segurança, espere até o momento da conexão quando você será solicitado a fornecer esta informação.

  • Utilize o botão OK para enviar todas as informações introduzidas e adicione o broker.

Para se conectar ao broker adicionado:

  1. Selecione o broker no painel de navegação do console de administração.
  2. Escolha Ações > Conectar ao broker.

Para consultar e/ou atualizar um broker em execução:

  1. Selecione o broker no painel de navegação do console de administração.
  2. Conecte-se a ele se ainda não estiver conectado.
  3. Escolha Ações > Consultar/atualizar broker.

Para alterar as informações especificadas nesta caixa de diálogo para determinado broker, você deve fazer o seguinte:

  1. Selecione o broker no painel de navegação.
  2. Desconecte o broker se já não estiver desconectado. O broker está desconectado se, no painel de navegação, o ícone aparecer da seguinte forma:
    Para desconectar um broker, escolha Ações > Desconectar do broker
  3. Escolha Ações > Propriedades para abrir a caixa de diálogo Propriedades do broker.
