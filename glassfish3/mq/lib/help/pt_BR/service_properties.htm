Propriedades de serviço

Utilize esta caixa de diálogo para especificar as propriedades do serviço do Message Queue. É possível alterar qualquer uma das seguintes propriedades sem interromper o serviço.

Número da porta Especifica o número da porta dinamicamente alocada para o serviço selecionado pelo Port Mapper. É possível clicar no botão de opção Estático e especificar um número de porta alternativo (disponível) para o serviço.

Estado do serviço Especifica o estado atual do serviço.

Número atual de segmentos alocados Especifica o número de segmentos alocado para este serviço.

Número atual de conexões Especifica o número de conexões alocado para este serviço.

Número mínimo de segmentos Especifica o número mínimo de segmentos que deve ser alocado do grupo de segmentos para este serviço.

Número máximo de segmentos Especifica o número máximo de segmentos que deve ser alocado do grupo de segmentos para este serviço.

Clique em OK para definir estes valores de configuração para o serviço selecionado.
