Assinaturas duráveis

Esta aba exibe uma lista de assinantes permanentes para o destino de tópico atual.

A tabela especifica para cada assinante o Client ID do assinante, o número de mensagens esperando para serem consumidas e o estado das assinaturas.

Para excluir um assinante permanente:

  1. Clique no assinante na tabela para selecionar tal assinante.
  2. Clique no botão Excluir. Observe que esta exclusão é imediata. Não é necessário clicar no botão OK.

Para limpar as mensagens de um assinante:

  1. Clique no assinante na tabela para selecionar tal assinante.
  2. Clique no botão Limpar. Observe que esta ação é imediata. Não é necessário clicar no botão OK.
