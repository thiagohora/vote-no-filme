3.0 연결 처리

연결 처리 탭을 사용하여 이 팩토리에서 만드는 연결에 관한 기본 정보를 지정합니다.

연결 유형 드롭 목록을 사용하여 연결 유형을 선택합니다.

브로커 호스트 이름 클라이언트에서 연결할 브로커를 호스트하는 시스템의 이름을 지정합니다.

브로커 호스트 포트 포트 매퍼 포트 번호를 지정합니다. 클라이언트는 포트 매퍼가 제공하는 정보를 사용하여 jms나 ssljms 중에서 원하는 서비스에 연결합니다.

브로커 서비스 포트 jms나 ssljms 서비스에 직접 연결하기 위해 사용할 포트의 번호를 지정합니다. 포트 매퍼를 우회하려면 브로커 서비스 포트를 사용합니다. 이 경우 클라이언트는 알려진 단일 포트를 사용하여 방화벽을 통해 연결할 수 있습니다.

HTTP URL 메시지 서비스 연결에 사용할 URL을 지정합니다. 기본값은 http://localhost/imq/tunnel입니다.
