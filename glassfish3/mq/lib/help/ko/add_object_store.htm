객체 저장소 추가

이 대화 상자를 사용하여 관리 도구의 탐색 창에 기존 객체 저장소를 추가합니다. 객체 저장소를 추가하고 나면 이 저장소에 연결하여 대상 객체와 연결 팩토리 객체를 추가할 수 있습니다.

객체 저장소에는 대상 객체나 연결 팩토리 객체와 같은 사용자가 만든 관리 객체가 포함되어 있습니다. 클라이언트 응용 프로그램에서 호출한 표준 JNDI 조회 코드를 통해 저장소의 내용에 액세스할 수 있습니다. 객체 저장소는 파일 기반 저장소일 수도 있고 LDAP 서버일 수도 있습니다. 파일 기반 저장소를 사용하려면 해당 폴더를 만들어야 합니다. LDAP 서버를 사용하려면 해당 서버를 구성하고 시작해야 합니다.

객체 저장소 레이블: 객체 저장소의 이름을 지정합니다. 이 이름은 관리 콘솔의 탐색 창에 표시됩니다.

JNDI 이름 지정 서비스 등록 정보: 이 영역의 컨트롤을 사용하면 이 객체 저장소의 JNDI 등록 정보 목록을 정의할 수 있습니다. 이름 드롭다운 목록에서 등록 정보 이름을 선택합니다. 등록 정보 이름을 선택한 다음 값 필드에 등록 정보 값을 입력합니다.

JNDI 응용 프로그램 프로그래밍 인터페이스에 대한 내용은 http://java.sun.com/products/jndi/1.2/javadoc/를 참조하십시오. java.naming.factory.initial and java.naming.provider.url 등록 정보를 설정해야 합니다.

초기 컨텍스트 등록 정보(java.naming.factory.initial)는 2개의 값을 가질 수 있습니다.

  • 파일 시스템 저장소: com.sun.jndi.fscontext.RefFSContextFactory
  • LDAP 서버: com.sun.jndi.ldap.LdapCtxFactory

또한 인증 정보도 지정해야 합니다. 하지만 보안이 염려되는 경우에는 연결할 때 해당 정보를 입력하라는 메시지가 표시될 때까지 기다려야 합니다.

  • 등록 정보 이름과 값을 지정한 다음 추가 버튼을 눌러 테이블에 등록 정보를 추가합니다.
  • 이미 추가한 등록 정보를 삭제하려면 테이블에서 등록 정보를 선택한 후 삭제 버튼을 누릅니다.
  • 항목을 모두 지우려면 지우기 버튼을 누릅니다.
  • 등록 정보를 재구성하려면 항목을 선택하고 값을 변경한 후 변경 버튼을 누릅니다.
  • 모든 등록 정보 설정을 완결하고 객체 저장소를 추가하려면 확인 버튼을 누릅니다.

기존 객체 저장소의 등록 정보를 변경하려면 다음을 수행합니다.

  1. 객체 저장소와 아직 연결되어 있다면 연결을 해제합니다. 탐색 창에서 객체 저장소 아이콘 모양이 이면 객체 저장소가 아직 연결되어 있다는 뜻입니다.
    객체 저장소 연결을 해제하려면 탐색 창에서 저장소를 선택하고 작업 > 객체 저장소에서 연결 해제를 선택합니다.
  2. 객체 저장소 등록 정보 대화 상자를 표시하려면 작업 > 등록 정보를 선택합니다.

객체 저장소에 대상 객체를 추가하려면 대상 폴더를 누른 후 작업 > 대상 객체 추가를 선택합니다.

객체 저장소에 연결 팩토리를 추가하려면 연결 팩토리 폴더를 누른 후 작업 > 연결 팩토리 객체 추가를 선택합니다.
