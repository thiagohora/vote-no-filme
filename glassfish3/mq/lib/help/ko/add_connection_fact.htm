연결 팩토리 객체 추가

이 대화 상자를 사용하여 연결 팩토리 객체를 구성하고 객체 저장소에 추가합니다.

연결 팩토리는 JMS 클라이언트에서 브로커에 연결할 때 사용되는 관리 대상 객체입니다. 이 객체는 연결이 동작하는 방식과 연결을 사용하는 클라이언트 런타임이 브로커에 액세스하는 방식을 지정하는 데에도 사용됩니다.

조회 이름 이 필드를 사용하여 연결 팩토리 객체의 JNDI 조회 이름을 지정합니다.

팩토리 유형 팩토리에서 만드는 연결의 유형을 지정하는 메뉴 항목을 ConnectionFactory, QueueConnectionFactory, TopicConnectionFactory 중에서 선택합니다. 메시지가 배포된 트랜잭션의 일부여야 하는 경우에는 XAConnectionFactory, XAQueueConnectionFactory, XATopicConnectionFactory 중 하나를 선택합니다.

읽기 전용 팩토리 연결 객체를 읽기 전용으로 만들려면 이 확인란을 누릅니다. 객체가 읽기 전용이면 클라이언트 응용 프로그램에서 API 호출을 통해 연결 팩토리 속성을 수정할 수 없습니다.

연결 팩토리 객체를 구성하면 팩토리에서 만드는 모든 연결에 공통된 등록 정보를 지정할 수 있습니다. 이 대화 상자에 있는 탭을 사용하면 연결 팩토리를 구성할 수 있습니다. 특정 탭에 대한 자세한 내용은 다음 링크를 참조하십시오.

  • 확인을 눌러 연결 팩토리 객체를 만듭니다.

연결 팩토리 객체의 등록 정보를 변경하려면 다음 작업을 수행합니다.

  1. 관리 콘솔의 탐색 창에서 연결 팩토리 폴더를 선택합니다.
  2. 관리 콘솔의 결과 창에서 연결 팩토리 객체를 선택합니다.
  3. 작업 > 등록 정보를 선택하여 연결 팩토리 객체 등록 정보 대화 상자를 가져옵니다.
