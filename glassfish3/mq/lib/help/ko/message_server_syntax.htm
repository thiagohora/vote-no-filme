메시지 서버 주소 구문

메시지 서버 주소를 지정하는 구문은 다음과 같습니다.

scheme://address_syntax

여기서 scheme 및 address_syntax는 다음 표에서 설명합니다.

방식 연결 서비스 설명 구문
mq jms
및
ssljms 		지정된 호스트와 포트의 Message Queue 포트 매퍼가 연결 요청을 처리하고 지정된 연결 서비스를 기준으로 포트를 동적으로 지정합니다. 포트 번호가 알려지면 Message Queue가 연결을 설정합니다. [hostName][:port][/serviceName]
기본값(jms 서비스만 해당):
hostName = localhost
port = 7676
serviceName = jms
mqtcp jms Message Queue는 지정된 호스트와 포트에 직접 tcp 연결을 설정하여 연결합니다. hostName:port/jms
mqssl ssljms Message Queue는 지정된 호스트와 포트에 직접 보안 ssl 연결을 설정하여 연결합니다. hostName:port/ssljms
htttp httpjms Message Queue는 지정된 Message Queue 터널 서블릿 URL에 직접 HTTP 연결을 설정합니다(브로커가 터널 서블릿에 액세스하도록 구성해야 합니다). HTTPtunnelServletURL
htttps httpsjms Message Queue는 지정된 Message Queue 터널 서블릿 URL에 직접 HTTPS 연결을 설정합니다(브로커가 터널 서블릿에 액세스하도록 구성해야 합니다). HTTPStunnelServletURL

 

다음 표는 메시지 서버 주소 구문이 적용되는 일반적인 몇 가지 예를 보여줍니다.

연결 서비스 브로커 호스트 포트 주소 예
지정 안 됨 지정 안 됨 지정 안 됨 기본값
(mq://localHost:7676/jms)
지정 안 됨 지정된 호스트 지정 안 됨 myBkrHost
(mq://myBkrHost:7676/jms)
지정 안 됨 지정 안 됨 지정된 포트 매퍼 포트 1012
(mq://localHost:1012/jms)
ssljms 로컬 호스트 기본 포트 매퍼 포트 mq://localHost:7676/ssljms
ssljms 지정된 호스트 기본 포트 매퍼 포트 mq://myBkrHost:7676/ssljms
ssljms 지정된 호스트 지정된 포트 매퍼 포트 mq://myBkrHost:1012/ssljms
jms 로컬 호스트 지정된 서비스 포트 mqtcp://localhost:1032/jms
ssljms 지정된 호스트 지정된 서비스 포트 mqssl://myBkrHost:1034/ssljms
httpjms N/A N/A http://websrvr1:8085/imq/tunnel
httpsjms N/A N/A https://websrvr2:8090/imq/tunnel
