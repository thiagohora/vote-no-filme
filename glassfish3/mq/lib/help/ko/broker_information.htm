브로커 등록 정보

이 대화 상자를 사용하여 관리 콘솔의 탐색 창에서 브로커의 표현을 변경할 수 있습니다. 이를 위해 브로커를 시작할 필요는 없습니다. 이 대화 상자의 필드를 편집하려면 브로커에서 관리 콘솔의 연결을 끊고 대화 상자를 다시 표시해야 합니다.

브로커 레이블 관리 콘솔의 탐색 창에 표시할 이 브로커의 이름을 지정합니다. 이 이름이 브로커의 인스턴스 이름과 같을 필요는 없습니다.

호스트 이 브로커를 호스트하는 시스템의 이름을 지정하거나 기본값 localhost를 그대로 둡니다.

기본 포트 포트 매퍼에 지정된 포트의 번호를 지정합니다. 호스트 이름과 기본 포트 번호를 모두 사용하여 브로커를 고유하게 식별합니다. 같은 서버에서 브로커 인스턴스를 두 개 실행하려면 각 브로커의 포트 매퍼에 각각 다른 포트 번호가 있어야 합니다.

사용자 이름 사용자 이름을 지정합니다. 관리자의 경우 기본값은 admin입니다. 이 정보는 브로커에 연결을 시도할 때 인증에 사용됩니다. 이 대화 상자에서 비밀번호를 입력하는 것은 안전하지 않습니다. 보안이 염려되는 경우 브로커에 연결하여 해당 정보를 입력하라는 메시지가 표시될 때까지 기다려야 합니다.

비밀번호 사용자 비밀번호를 지정합니다. 관리자의 경우 기본값은 admin입니다. 이 정보는 브로커에 연결을 시도할 때 인증에 사용됩니다. 이 대화 상자에서 암호를 입력하는 것은 안전하지 않습니다. 보안이 염려되는 경우 브로커에 연결하여 해당 정보를 입력하라는 메시지가 표시될 때까지 기다려야 합니다.

브로커에 다시 연결하려면 다음 작업을 수행합니다.

  1. 관리 콘솔의 탐색 창에서 브로커를 선택합니다.
  2. 작업 > 브로커에 연결을 선택합니다.
