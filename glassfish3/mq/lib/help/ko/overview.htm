개요

관리 콘솔의 컨트롤을 통해 하나 이상의 Message Queue 브로커 및 객체 저장소와 통신할 수 있습니다.

관리 콘솔은 다음과 같이 다섯 개 창으로 나뉩니다.

  • 1 - 메뉴 표시줄
  • 2 - 도구 모음
  • 3 - 탐색 창
  • 4 - 결과 창
  • 5 - 상태 창

메뉴 표시줄의 메뉴나 도구 모음의 아이콘을 사용하면 탐색 창과 결과 창에서 선택한 요소에 작업을 할 수 있습니다. 상태 창에는 최근 완료된 작업에 대한 정보가 표시됩니다.

탐색 창에는 연결된 객체 저장소와 브로커가 트리 보기로 표시됩니다. 객체 저장소에는 대상(대기열이나 주제) 또는 연결 팩토리에 해당하는 관리 대상 객체가 포함됩니다. 브로커에는 서비스와 물리적 대상의 배열이 포함됩니다.

  • 객체 저장소에 있는 대상을 보려면 대상 폴더를 클릭합니다. 그러면 내용이 결과 창에 표시됩니다.
  • 객체 저장소의 연결 팩토리에 대한 정보를 보려면 연결 팩토리 폴더를 누릅니다. 각 연결 팩토리의 조회 이름 및 유형이 결과 창에 표시됩니다.
  • 브로커에서 사용할 수 있는 서비스에 대한 정보를 보려면 서비스 폴더를 누릅니다. 서비스, 각 서비스의 포트 번호 및 상태 목록이 결과 창에 표시됩니다.
  • 특정 서비스에 대한 정보를 보거나 변경하려면 결과 창에서 서비스를 선택한 후 작업 > 등록 정보를 선택합니다.
  • 브로커에서 물리적 대상에 대한 정보를 보려면 대상 폴더를 누릅니다. 해당 브로커에서 생성된 대상의 목록이 대상 유형 그리고 대기열의 경우 전달 정책과 함께 표시됩니다.
  • 특정 대상에 대한 정보를 보거나 변경하려면 결과 창에서 대상을 선택한 후 작업 > 등록 정보를 선택합니다.

객체 저장소 또는 브로커의 등록 정보를 변경하려면 먼저 연결을 끊어야 합니다.

관리 콘솔을 사용하면 그래픽 인터페이스를 통해 Message Queue 관리의 기본 기능에 액세스할 수 있습니다. 하지만 적절한 Message Queue 명령 유틸리티를 사용하면 더 많은 기능을 사용할 수 있습니다. 자세한 내용은 Message Queue 관리자 설명서를 참조하십시오.
