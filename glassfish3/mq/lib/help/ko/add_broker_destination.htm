브로커 대상 추가

이 대화 상자를 사용하여 브로커에 물리적 대상을 만들고 구성합니다. 브로커에서 대상은 받는 메시지가 메시지 사용자에게 라우팅되기 전에 마샬링되는 브로커의 물리적인 메모리 위치를 의미합니다.

Message Queue 메시징은 두 단계의 메시지 전달을 통해 구현됩니다. 먼저 메시지가 생성자에서 브로커 상의 대상으로 전달됩니다. 그런 다음 브로커 상의 대상에서 한 명 이상의 사용자에게 메시지가 전달됩니다.

물리적 대상을 만들지 않은 경우에는 동적으로 생성하도록 브로커를 구성할 수 있습니다. 이렇게 자동 생성된 대상은 더 이상 필요가 없어지면 브로커가 자동으로 완전 삭제합니다.

물리적 대상을 구성하려면 다음 속성 값을 설정합니다.

대상 이름 물리적 대상의 이름을 지정합니다.

대상 유형 대상 유형(대기열 또는 주제)을 지정합니다. 대기열 또는 주제 라디오 버튼을 누릅니다.

최대 메시지 수 해당 대상에서 허용되는 사용되지 않은 메시지의 최대 수를 지정합니다.

메시지의 최대 전체 크기(바이트) 해당 대상에서 사용되지 않은 메시지에 대해 허용되는 최대 전체 메모리 양을 지정합니다. 드롭다운 목록에서 바이트, KB 및 MB 중 단위를 선택합니다.

메시지당 최대 크기(바이트) 대상에서 한 메시지에 허용되는 최대 크기를 지정합니다. 드롭다운 목록에서 바이트, KB 및 MB 중 하나의 단위를 선택합니다.

최대 생성자 수 해당 대상에서 허용되는 생성자의 최대 수를 지정합니다.

최대 활성 사용자 수 대기열 대상에만 적용됩니다. 해당 대상으로부터의 로드 균형 조정 전달에서 활성 상태가 될 수 있는 사용자의 최대 수를 지정합니다.

최대 백업 사용자 수 대기열 대상에만 적용됩니다. 해당 대상으로부터의 로드 균형 조정 전달 중에 활성 사용자가 실패할 경우 이 사용자를 대체할 수 있는 백업 사용자의 최대 수를 지정합니다.

대상의 현재 상태에 대한 정보를 얻거나 대상의 등록 정보를 변경하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 작업 > 등록 정보를 선택합니다.

대상에서 메시지를 제거하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 작업 > 메시지 제거를 선택합니다.

대상을 삭제하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 편집 > 삭제를 선택합니다.

대상을 일시 중지하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 작업 > 생성자 일시 중지 | 사용자 일시 중지 | 모두 일시 중지를 선택합니다.

일시 중지된 대상을 다시 시작하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 작업 > 다시 시작을 선택합니다.
