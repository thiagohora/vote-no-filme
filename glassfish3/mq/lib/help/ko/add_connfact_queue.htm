QueueBrowsers 및 ServerSessions

이 탭을 사용하여 클라이언트 런타임의 대기열 찾아보기를 구성하고 응용 프로그램 서버에서 실행되는 세션의 동작을 지정합니다.

대기열 브라우저 검색 크기 클라이언트 런타임에서 대기열 대상의 내용을 찾아볼 때 한 번에 검색하는 메시지의 최대 수를 지정합니다. 기본값은 1000입니다.

대기열 브라우저 검색 시간 초과(밀리초) 클라이언트 런타임에서 대기열 대상의 내용을 찾아볼 때 메시지를 검색하기 위해 기다리는 최대 시간을 지정합니다. 이 시간이 초과되면 예외가 발생합니다. 기본값은 60000밀리초입니다.

서버 세션에 최대 메시지 로드 Message Queue ConnectionConsumer가 ServerSession의 세션에 maxMessages까지 로드하도록 지정합니다. Message Queue ConnectionConsumer가 한 번에 한 메시지만을 로드하도록 지정하려면 이 확인란을 선택하지 않은 상태로 둡니다. 자세한 내용은 JMS 사양의 응용 프로그램 서버 기능 섹션을 참조하십시오.
