대상 객체 등록 정보

이 대화 상자를 사용하여 대상 객체의 등록 정보를 변경합니다.

읽기 전용 이 확인란을 선택하면 객체의 읽기 전용 상태를 토글할 수 있습니다. 객체를 읽기 전용으로 표시하면 클라이언트 응용 프로그램은 API 호출을 통해 관리 대상 객체의 속성을 수정할 수 없습니다.

대상 이름 이 필드를 사용하면 관리 대상 객체에 해당하는 브로커의 물리적 대상 이름을 지정할 수 있습니다. 이름을 지정하지 않으면 기본 이름인 Untitled_Destination_Object가 사용됩니다.

대상 설명 이 필드를 사용하면 대상 객체에 대한 유용한 설명을 입력할 수 있습니다.

대화 상자에 지정한 값을 사용하여 대상 객체를 재구성하려면 확인 버튼을 누릅니다.
