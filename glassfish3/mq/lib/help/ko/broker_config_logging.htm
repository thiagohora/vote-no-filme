로깅

이 탭을 사용하여 브로커 인스턴스의 로깅을 구성합니다.

로그 수준 임의의 출력 채널에 기록할 수 있는 로깅 출력의 수준을 지정합니다. 심각도가 낮아지는 순으로 나타낸 범주로는 ERROR, WARNING, INFO가 있습니다. 드롭 목록에서 범주를 선택합니다. 범주를 선택하면 로거는 지정된 범주 이상에 해당하는 데이터를 씁니다. 예를 들어 INFO를 선택한 경우에는 세 범주의 모든 데이터를 얻게 됩니다. ERROR를 선택하면, ERROR 메시지만 얻게 됩니다. 기본값은 INFO입니다.

로그 롤오버 크기 출력이 새 로그 파일로 롤오버되는 로그 파일의 크기를 지정합니다. 드롭 목록에서 바이트, KB, MB 중 단위를 선택합니다. 로그 파일 크기 기준의 롤오버를 지정하지 않으려면 제한 없음 라디오 버튼을 선택합니다. 기본값은 제한 없음입니다.

로그 롤오버 간격 출력이 새 로그 파일로 롤오버될 때 로그 파일의 표시 시간(초)을 지정합니다. 파일 표시 시간 기준의 롤오버를 지정하지 않으려면 제한 없음 라디오 버튼을 선택합니다. 기본값은 604800입니다.
