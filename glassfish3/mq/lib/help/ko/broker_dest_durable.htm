영구 가입

이 탭에는 현재 주제 대상에 대한 영구 가입자의 목록이 표시됩니다.

테이블에서 각 가입자의 ClientID와 사용 대기 중인 메시지의 수, 영구 가입 상태를 지정합니다.

영구 가입자를 삭제하려면 다음 작업을 수행합니다.

  1. 테이블에서 가입자를 눌러 선택합니다.
  2. 삭제 버튼을 누릅니다. 삭제 버튼을 누르면 바로 삭제되기 때문에 확인 버튼을 누를 필요가 없습니다.

가입자의 메시지를 제거하려면 다음 작업을 수행합니다.

  1. 테이블에서 가입자를 눌러 선택합니다.
  2. 제거 버튼을 누릅니다. 제거 버튼을 누르면 바로 제거되기 때문에 확인 버튼을 누를 필요가 없습니다.
