브로커 대상 등록 정보 >
기본

이 탭에는 브로커 대상에 관한 정보가 표시됩니다. 브로커 대상 추가 대화 상자에서 대상에 대해 정의한 등록 정보를 이 탭을 사용하여 변경할 수도 있습니다.

대상 이름 물리적 대상의 이름을 지정합니다.

대상 유형 대상 유형(대기열 또는 주제)을 지정합니다.

대상 상태 대상이 활성 또는 중지된 상태(중지된 생성자, 중지된 사용자 또는 둘 다)인지 지정합니다.

현재 메시지 수 현재 해당 대상에 있으면서 사용되지 않은 메시지의 수를 지정합니다.

현재 메시지 전체 크기(바이트) 현재 해당 대상에서 사용되지 않은 메시지가 사용하는 전체 메모리의 양을 지정합니다.

현재 생성자 수 현재 해당 대상과 연관되어 있는 생성자의 수를 지정합니다.

현재 사용자 수 현재 해당 대상과 연관되어 있는 사용자의 수를 지정합니다. 여기에는 비-durables와 활성/비활성 durables(주제에만 적용)가 포함됩니다.

현재 활성 사용자 수 현재 해당 대상(대기열만 해당)과 연관되어 있는 활성 사용자의 수를 지정합니다.

현재 백업 사용자 수 현재 해당 대상(대기열만 해당)과 연관되어 있는 백업 사용자의 수를 지정합니다.

최대 메시지 수 해당 대상에 허용된 사용되지 않은 메시지의 최대 수를 지정합니다.

메시지의 최대 전체 크기(바이트) 해당 대상에서 사용되지 않은 메시지가 사용할 수 있는 전체 메모리 양의 최대값을 지정합니다. 드롭다운 목록에서 바이트, KB, MB 중 하나의 단위를 선택합니다.

메시지당 최대 크기(바이트) 해당 대상에서 허용되는 단일 메시지의 최대 크기를 지정합니다. 드롭다운 목록에서 바이트, KB, MB 중 하나의 단위를 선택합니다.

최대 생성자 수 해당 대상에서 허용된 최대 생성자 수를 지정합니다.

최대 활성 사용자 수 대기열 대상에만 적용됩니다. 해당 대상으로부터의 로드 균형 조정 전달에서 활성 상태가 될 수 있는 최대 사용자 수를 지정합니다.

최대 백업 사용자 수 대기열 대상에만 적용됩니다. 해당 대상으로부터의 로드 균형 조정 전달 중에 어떤 활성 사용자가 실패할 경우 그 활성 사용자를 대신할 백업 사용자의 최대 수를 지정합니다.

제한 동작 해당 대상에 대한 메모리 제한 임계값에 도달할 경우에 브로커가 응답하는 방법을 지정합니다. 값은 다음과 같습니다.

  • FLOW_CONTROL - 생성자 수를 줄입니다.
  • REMOVE_OLDEST - 가장 오래된 메시지를 삭제합니다.
  • REMOVE_LOW_PRIORITY - 메시지의 보존 기간을 기준으로 우선 순위가 가장 낮은 메시지를 삭제합니다(생성자 클라이언트는 메시지 삭제에 대한 알림 메시지를 받지 않음).
  • REJECT_NEWEST - 최신 메시지를 거부합니다(생성자 클라이언트는 지속성 메시지가 거부되는 경우 예외 메시지를 받지만 비지속성 메시지가 거부되는 경우 알림 메시지를 받지 않음).

사용 불능 메시지 대기열 사용 해당 대상에 대한 사용 불능 메시지 대기열의 사용을 지정합니다.

전달할 수 없다고 판단되는 메시지는 사용 불능 메시지 대기열이라고 부르는 특정 대기열에 자동으로 배치됩니다. 이 대기열의 메시지는 원본 대상을 포함한 원본 헤더를 보존하고 사용 불능 메시지가 된 이유를 설명하는 정보가 메시지의 등록 정보에 추가됩니다. 관리자나 개발자가 이 대기열에 액세스하여 메시지를 제거하거나 그 메시지가 이 대기열에 있는 이유를 확인할 수 있습니다.

사용 불능 메시지 대기열은 시스템이 자동으로 생성하고 mq.sys.dmq로 명명됩니다.

대상에서 메시지를 제거하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 작업 > 메시지 제거를 선택합니다.

대상을 삭제하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 편집 > 삭제를 선택합니다.

대상을 일시 중지하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 작업 > 생성자 일시 중지 | 사용자 일시 중지 | 모두 일시 중지를 선택합니다.

일시 중지된 대상을 다시 시작하려면 다음 작업을 수행합니다.

  1. 콘솔의 결과 창에서 대상을 선택합니다.
  2. 작업 > 다시 시작을 선택합니다.
