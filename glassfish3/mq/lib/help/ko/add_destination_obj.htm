대상 객체 추가

이 대화 상자를 사용하여 객체 저장소에 대상 객체를 추가합니다.

대상 관리 객체는 브로커의 물리적 대상(대기열 또는 주제)에 대한 공급자의 영향을 받지 않는 표현입니다.

조회 이름 이 필드를 사용하여 관리 객체의 JNDI 조회 이름을 지정합니다. 이 이름은 대상 이름과 같을 수 있습니다. 유일한 제약은 객체 저장소의 컨텍스트에서 고유해야 한다는 것입니다.

대상 유형 대기열과 주제라는 두 대상 유형 중 하나에 해당하는 라디오 버튼을 누릅니다.

읽기 전용 객체를 읽기 전용으로 만들려면 이 확인란을 누릅니다. 객체가 읽기 전용이면 클라이언트 응용 프로그램은 API 호출을 통해 관리 대상 객체의 속성을 수정할 수 없습니다.

대상 이름 이 필드를 사용하면 관리 대상 객체에 해당하는 브로커의 물리적 대상 이름을 지정할 수 있습니다. 이름을 지정하지 않으면 기본 이름인 Untitled_Destination_Object가 사용됩니다.

대상 설명 이 필드를 사용하면 대상 객체에 대한 유용한 설명을 입력할 수 있습니다.

  • 모든 필드를 기본값으로 재설정하려면 기본값으로 재설정 버튼을 누릅니다.
  • 대상 객체를 만들려면 확인을 누릅니다.

대상 객체의 등록 정보를 변경하려면 다음 작업을 수행합니다.

  1. 관리 콘솔의 탐색 창에서 대상 폴더를 선택합니다.
  2. 관리 콘솔의 결과 창에서 대상 객체를 선택합니다.
  3. 작업 > 등록 정보를 선택하여 대상 객체 등록 정보 대화 상자를 표시합니다.
