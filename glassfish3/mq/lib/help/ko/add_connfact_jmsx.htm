JMSX 등록 정보

이 탭을 사용하여 Message Queue에서 설정해야 하는 JMS 정의 등록 정보를 지정합니다.

Message Queue에서는 많은 JMS 정의 속성을 지원하는 데 이것을 지원하려면 자원이 필요합니다. 자원을 절약하려면 클라이언트 응용 프로그램에 실제로 필요한 등록 정보를 파악하고 그 등록 정보의 확인란을 선택하여 설정할 수 있게 해야 합니다.

JMSXAppID 메시지 등록 정보 사용 Message Queue가 생성된 메시지에 대해 JMS 정의 등록 정보인 JMSXAppID(메시지를 보내는 응용 프로그램의 아이디)를 설정할 것인지 여부를 지정합니다. 기본값은 false입니다.

JMSXUserID 메시지 등록 정보 사용 Message Queue가 생성된 메시지에 대해 JMS 정의 등록 정보인 JMSXUserID(메시지를 보내는 사용자의 아이디)를 설정할 것인지 여부를 지정합니다. 기본값은 false입니다.

JMSXProducerTXID 메시지 등록 정보 사용 Message Queue가 생성된 메시지에 대해 JMS 정의 등록 정보인 JMSXProducerTXID(메시지를 생성한 트랜잭션의 아이디)를 설정할 것인지 여부를 지정합니다. 기본값은 false입니다.

JMSXConsumerTXID 메시지 등록 정보 사용 Message Queue가 사용된 메시지에 대해 JMS 정의 등록 정보인 JMSXConsumerTXID(메시지를 사용한 트랜잭션의 아이디)를 설정할 것인지 여부를 지정합니다. 기본값은 false입니다.

JMSXRcvTimestamp 메시지 등록 정보 사용 Message Queue가 사용된 메시지에 대해 JMS 정의 등록 정보인 JMSXRcvTimestamp(메시지가 사용자에게 전달된 시간)를 설정할 것인지 여부를 지정합니다. 기본값은 false입니다.
