基本

使用此选项卡，可以指定或浏览代理实例的基本配置信息。

版本 指定代理的版本。

实例名 指定默认名 imqbroker 或在启动代理时分配的名称。

主端口 为代理的端口映射器端口指定端口号。端口映射器负责将端口动态分配给 Message Queue 服务。此端口号应与启动代理时指定的端口号相同。

自动创建主题 指定每当消息用户或消息生产者尝试访问不存在的目的地时是否允许代理创建主题目的地。

自动创建队列 指定每当消息用户或消息生产者尝试访问不存在的目的地时是否允许代理创建队列目的地。

自动创建队列的最大活动用户数 指定可以在从队列目的地的负载平衡传送中处于活动状态的最大用户数。

自动创建队列的最大备份用户数 指定在从队列目的地的负载平衡传送期间，在任何活动用户失败时可以代替这些活动用户的最大备份用户数。
