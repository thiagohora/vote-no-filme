添加目的地对象

使用此对话框，可以将目的地对象添加到对象存储中。

目的地管理对象是代理中物理目的地（队列或主题）独立于提供者的表示。

查找名称 使用此字段可以指定被管理对象的 JNDI 查找名称。此名称可与目的地名称相同，唯一的限制就是此名称在对象存储环境中必须唯一。

目的地类型 单击与目的地类型对应的单选按钮：队列或主题。

只读 单击此复选框，可以将对象设置为只读。如果对象为只读对象，客户机应用程序将无法通过 API 调用修改被管理对象的属性。

目的地名 使用此字段，可以指定代理中与此被管理对象相对应的物理目的地的名称。如果您不指定名称，则使用默认名称 Untitled_Destination_Object。

目的地说明 使用此字段，可以输入此目的地对象的有用说明。

  • 单击“重置为默认值”按钮，将所有字段重置为默认值。
  • 单击“确定”创建目的地对象。

更改目的地对象属性的步骤：

  1. 在管理控制台的导航窗格中选择 Destinations 文件夹。
  2. 在管理控制台的结果窗格中选择目的地对象。
  3. 选择“操作”>“属性”打开“目的地对象属性”对话框。
