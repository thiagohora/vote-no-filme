连接工厂对象属性

使用此对话框，可以重新配置使用“添加连接工厂”对话框设置的连接工厂属性。

只读 单击此复选框，可以将工厂连接对象设置为只读。如果对象为只读对象，客户机应用程序将无法通过 API 调用修改连接工厂属性。

通过重新配置连接工厂对象，用户可以重新定义此工厂生成的所有连接的通用属性。可以使用此对话框中显示的选项卡重新配置连接工厂。有关特定选项卡的信息，请单击下列链接之一。
