Proprietà oggetto di destinazione

Utilizzare questa finestra di dialogo per modificare le proprietà dell'oggetto di destinazione.

Sola lettura Fare clic su questa casella di controllo per commutare lo stato di sola lettura dell'oggetto. Se un oggetto è contrassegnato come sola lettura, le applicazioni client non potranno modificare gli attributi dell'oggetto amministrato tramite le chiamate api.

Nome destinazione Utilizzare questo campo per specificare il nome della destinazione fisica sul broker a cui corrisponde tale oggetto amministrato. Se non è specificato un nome, viene utilizzato il nome predefinito di Untitled_Destination_Object .

Descrizione della destinazione Utilizzare questo campo per inserire una descrizione utile di questo oggetto di destinazione.

Fare clic sul pulsante OK per riconfigurare l'oggetto di destinazione utilizzando i valori specificati nella finestra di dialogo.
