Log

Utilizzare questa scheda per configurare la registrazione per una istanza del broker.

Livello di log Specifica il livello di output di registrazione che può essere scritto in ogni canale di output. Le categorie, in ordine decrescente di gravità, sono ERRORE, ATTENZIONE, INFORMAZIONI. Utilizzare l'elenco a discesa per selezionare una categoria. Quando viene selezionata una categoria, il log scrive i dati relativi alla categoria specificata e a tutte le categorie superiori. Ad esempio, se si seleziona INFORMAZIONI, verranno acquisiti i dati in tutte e tre le categorie. Se si seleziona ERRORE, verranno acquisiti soltanto messaggi di ERRORE . L'impostazione predefinita è INFORMAZIONI.

Rollover del log Dimensioni Specifica le dimensioni del file di registro a cui l'output esegue un rollover a un nuovo file di registro. Utilizzare l'elenco a discesa per selezionare le unità di misura: byte, kilobyte, o megabyte. Selezionare il pulsante di scelta Illimitato per specificare nessun rollover basato sulle dimensioni del file di registro. L'opzione di selezione predefinita è Illimitato.

Intervallo di rollover del log Specifica l'età, in secondi, del file di registro quando l'output esegue un rollover a un nuovo file di registro. Selezionare il pulsante di scelta Illimitato per specificare nessun rollover basato sull'età del file. Il valore predefinito è 604800.
