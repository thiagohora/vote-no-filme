Messaggistica

Utilizzare questa scheda per configurare la capacità di messaggistica e le dimensioni del messaggio per questa istanza del broker.

Numero massimo di messaggi nel sistema Specifica il numero massimo di messaggi nella memoria e nel disco (a causa dello swap). Ulteriori messaggi saranno rifiutati. L'impostazione predefinita è Illimitato.

Byte messaggio totali massimi nel sistema Specifica le dimensioni totali massime (in byte, Kbyte, o Mbyte) dei messaggi nella memoria e nel disco (a causa dello swap). Ulteriori messaggi saranno rifiutati. L'impostazione predefinita è Illimitato. Utilizzare l'elenco a discesa per specificare le unità di misura utilizzate per calcolare le dimensioni: byte, kilobyte o megabyte.

Dimensioni massime messaggio Specifica le dimensioni massime consentite (in byte, Kbyte, o Mbyte) del corpo del messaggio. Ogni messaggio eccedente tali dimensioni sarà rifiutato. Il valore predefinito è 70 megabyte. Utilizzare l'elenco a discesa per specificare le unità di misura utilizzate per calcolare le dimensioni: byte, kilobyte, o megabyte.
