Di base

Utilizzare questa scheda per specificare o consultare informazioni di configurazione di base per una istanza del broker.

Versione Specifica la versione del broker.

Nome istanza Specifica il nome predefinito imqbroker oppure il nome che è stato assegnato all'avvio del broker.

Porta principale Specifica il numero per la porta portmapper del broker. Il portmapper è responsabile dell'assegnazione dinamica delle porte ai servizi di Message Queue. Questo dovrebbe essere lo stesso numero che è stato specificato all'avvio del broker.

Creazione automatica argomenti Specifica se il broker può creare una destinazione per l'argomento ogni volta che un consumatore o un produttore di messaggi tenta di accedere a una destinazione inesistente.

Creazione automatica code Specifica se il broker può creare una destinazione per la coda ogni volta che un consumatore o un produttore di messaggi tenta di accedere a una destinazione inesistente.

Numero massimo di consumatori attivi della coda creata automaticamente Specifica il numero massimo di consumatori che possono essere attivi nel recapito con carico bilanciato da questa una destinazione.

Numero massimo di consumatori backup della coda creata automaticamente Specifica il numero massimo di consumatori di backup che possono sostituire i consumatori attivi, in caso di errore durante la distribuzione bilanciata di questi ultimi dalla destinazione.
