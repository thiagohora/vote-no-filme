Aggiungi broker

Utilizzare questa finestra di dialogo per creare la rappresentazione di un broker nel riquadro di navigazione della console di amministrazione. Per eseguire questa operazione, non è necessario avviare il broker. Tuttavia, è necessario avviare il broker prima di connetterlo.

Etichetta del broker Specifica il nome che di desidera visualizzare per questo broker nel riquadro di navigazione della console di amministrazione. Tale nome non deve essere lo stesso del nome dell'istanza del broker.

Host Specifica il nome del computer che ospita il broker o mantiene i valori predefiniti di localhost.

Porta principale Specifica il numero della porta assegnata al portmapper. Il valore predefinito è 7676. La combinazione tra il nome host e il numero della porta principale identifica in modo univoco il broker. Per eseguire due istanze di un broker sullo stesso server, il portmapper di ciascun broker deve avere un numero di porta diverso.

Nome utente Specifica il nome utente. Il valore predefinito è admin per l'amministratore. Questa informazione viene utilizzata per eseguire l'autenticazione quando si tenta la connessione al broker. Tuttavia, l'immissione di tale valore nella finestra di dialogo non è una procedura sicura. Se la sicurezza è una priorità, si consiglia di attendere il momento della connessione e la visualizzazione di un messaggio che richiede questa informazione.

Password Specifica la password dell'utente. Il valore predefinito è admin per l'amministratore. Questa informazione viene utilizzata per eseguire l'autenticazione quando si tenta la connessione al broker. Tuttavia, l'immissione di tale valore nella finestra di dialogo non è una procedura sicura. Se la sicurezza è una priorità, si consiglia di attendere il momento della connessione e la visualizzazione di un messaggio che richiede questa informazione.

  • Utilizzare il pulsante OK per eseguire il commit di tutte le informazioni inserite e aggiungere il broker.

Per connettersi al broker aggiunto:

  1. Selezionare il broker nel riquadro di navigazione della console di amministrazione.
  2. Selezionare Azioni > Connetti al broker.

Per ricercare e/o aggiornare un broker in esecuzione:

  1. Selezionare il broker nel riquadro di navigazione della console di amministrazione.
  2. Se non è già connesso, connetterlo.
  3. Selezionare Azioni > Ricerca/Aggiorna il broker.

Per modificare le informazioni specificate in questa finestra di dialogo per un broker in particolare, è necessario eseguire la procedura seguente:

  1. Selezionare il broker nel riquadro di navigazione.
  2. Se non è disconnesso, disconnettere il broker. Un broker risulta disconnesso se l'icona nel riquadro di navigazione appare così:
    Per disconnettere un broker, scegliere Azioni > Disconnetti dal broker
  3. Scegliere Azioni > Proprietà per aprire la finestra di dialogo Proprietà del broker.
