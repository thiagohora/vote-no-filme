Proprietà JMSX

Utilizzare questa scheda per specificare quali proprietà definite JMS devono essere impostate da Message Queue.

Message Queue supporta un numero di attributi definiti JMS, al costo delle risorse. Per risparmiare risorse, è necessario determinare quali proprietà siano realmente necessarie all'applicazione del client e consentire l'impostazione di tali proprietà facendo clic sull'apposita casella di controllo.

Abilita proprietà messaggio JMSXAppID Specifica se Message Queue deve impostare la proprietà definita JMS, JMSXAppID (identità dell'applicazione che invia il messaggio), sui messaggi prodotti. Predefinito: false

Abilita proprietà messaggio JMSXUserID Specifica se Message Queue deve impostare la proprietà definita JMS, JMSXUserID(identità dell'utente che invia il messaggio), sui messaggi prodotti. Predefinito: false

Abilita proprietà messaggio JMSXProducerTXID Specifica se Message Queue deve impostare la proprietà definita JMS, JMSXProducerTXID (identificatore della transazione durante la quale tale messaggio è stato prodotto), sui messaggi prodotti. Predefinito: false

Abilita JMSXConsumerTXID Proprietà messaggio Specifica se Message Queue deve impostare la proprietà definita JMS, JMSXConsumerTXID (identificatore della transazione durante la quale il messaggio è stato utilizzato) sui messaggi utilizzati. Predefinito: false

Abilita proprietà messaggio JMSXRcvTimestamp Specifica se Message Queue deve impostare la proprietà definita JMS, JMSXRcvTimestamp (l'orario in cui il messaggio è stato spedito al consumatore), sui messaggi utilizzati. Predefinito: false
