オブジェクトストアのプロパティ

このダイアログは、管理コンソールのナビゲーションペインに追加したオブジェクトストアのプロパティを追加、削除、または更新するために使用します。このダイアログのフィールドを編集するには、オブジェクトストアから管理コンソールを切断して、ダイアログを表示し直す必要があります。

JNDI ネーミングサービスプロパティこのエリアのコントロールを使用して、オブジェクトストアに関する JNDI プロパティのリストを定義します。プロパティ名は、「名前」ドロップダウンリストから選択します。プロパティ名を選択すると、「値」フィールドが表示されプロパティ値を入力することができます。

JNDI のアプリケーションプログラミングインタフェースの詳細は、http://java.sun.com/products/jndi/1.2/javadoc/ を参照してください。java.naming.factory.initial と java.naming.provider.url プロパティを設定する必要があります。

初期のコンテキストプロパティ (java.naming.factory.initial) は、次の 2 つの値のいずれかです。

  • ファイルシステムストア:com.sun.jndi.fscontext.RefFSContextFactory
  • LDAP サーバー:com.sun.jndi.ldap.LdapCtxFactory

また、認証情報の入力も必要です。ただし、セキュリティについて考慮する場合は、接続時にこの情報を要求されるまで入力しないようにすることもできます。

  • プロパティの名前と値を指定したら、「追加」ボタンをクリックして、テーブルにプロパティを追加します。
  • 既存のプロパティを削除するには、テーブル内のプロパティを選択して「削除」ボタンをクリックし、プロパティを削除します。
  • すべてのエントリを消去するには、「クリア」ボタンを使用します。
  • プロパティを設定し直すには、エントリを選択して値を変更し、「変更」ボタンを使用します。
  • すべてのプロパティの設定を確定するには、「OK」ボタンを使用します。

再設定した後でオブジェクトストアにもう一度接続するには、ナビゲーションペインでオブジェクトストアを選択し、「アクション」→「オブジェクトストアに接続」を選択します。
