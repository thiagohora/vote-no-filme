Broker hinzufügen

Mit Hilfe dieses Dialogfelds können Sie im Navigationsbereich der Administrationskonsole die Darstellung eines Brokers erstellen. Der Broker muss dazu nicht gestartet werden. Dieser muss erst gestartet werden, wenn Sie eine Verbindung zu ihm herstellen möchten.

Broker-Beschriftung: Gibt den Namen an, unter dem dieser Broker im Navigationsbereich der Administrationskonsole angezeigt werden soll. Dieser Name muss nicht mit dem Instanznamen des Brokers identisch sein.

Host: Gibt den Namen des Computers an, auf dem dieser Broker gehostet ist, oder behält den Standardwert localhost bei.

Primärer Anschluss: Gibt die Nummer des Anschlusses an, der dem Portmapper zugewiesen ist. Die Standardnummer lautet 7676. Anhand der Kombination aus Hostnamen und Nummer des primären Anschlusses kann der Broker eindeutig identifiziert werden. Wenn auf ein und demselben Server zwei Instanzen eines Brokers ausgeführt werden sollen, müssen die Portmapper der Broker unterschiedliche Anschlussnummern haben.

Benutzername: Gibt Ihren Benutzernamen an. Das Standard-Benutzerpasswort für den Administrator lautet admin. Diese Angabe wird verwendet, um Sie beim Verbindungsaufbau mit dem Broker zu authentifizieren. Die Eingabe Ihres Passworts in dieses Dialogfeld ist nicht sicher. Wenn Sicherheit ein Thema für Sie ist, sollten Sie mit der Eingabe dieser Information warten, bis Sie beim Verbindungsaufbau dazu aufgefordert werden.

Passwort: Gibt Ihr Benutzerpasswort an. Das Standard-Benutzerpasswort für den Administrator lautet admin. Diese Angabe wird verwendet, um Sie beim Verbindungsaufbau mit dem Broker zu authentifizieren. Die Eingabe Ihres Passworts in dieses Dialogfeld ist nicht sicher. Wenn Sicherheit ein Thema für Sie ist, sollten Sie mit der Eingabe dieser Information warten, bis Sie beim Verbindungsaufbau dazu aufgefordert werden.

  • Klicken Sie auf OK, um alle eingegebenen Informationen zu bestätigen und den Broker hinzuzufügen.

So stellen Sie eine Verbindung zu dem von Ihnen hinzugefügten Broker her:

  1. Wählen Sie im Navigationsbereich der Administrationskonsole den Broker aus.
  2. Wählen Sie den Befehl Aktionen > Mit Broker verbinden.

So können Sie eine Abfrage an einen laufenden Broker senden und/oder ihn aktualisieren:

  1. Wählen Sie im Navigationsbereich der Administrationskonsole den Broker aus.
  2. Stellen Sie, falls noch nicht geschehen, eine Verbindung zum Broker her.
  3. Wählen Sie Aktionen > Broker-Abfrage/Aktualisierung.

Wenn Sie die Informationen für einen bestimmten Broker, die Sie in diesem Dialogfeld eingegeben haben, ändern möchten, gehen Sie wie folgt vor:

  1. Wählen Sie im Navigationsbereich den Broker aus.
  2. Trennen Sie die Verbindung zum Broker, falls Sie noch verbunden sind. Wenn die Verbindung zu einem Broker getrennt ist, sieht dessen Symbol im Navigationsbereich wie folgt aus:
    Die Verbindung zum Broker können Sie mit dem Befehl Aktionen > Verbindung mit Broker trennen trennen.
  3. Wählen Sie den Befehl Aktionen > Eigenschaften, um das Dialogfeld Broker-Eigenschaften zu öffnen.
