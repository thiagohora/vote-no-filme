Eigenschaften Broker-Ziel

In diesem Dialogfeld können Sie sich den aktuellen Status eines Broker-Ziels anzeigen lassen und die für dieses Ziel festgelegten Eigenschaften ändern.

Auf der Registerkarte Grundlagen finden Sie Informationen über den aktuellen Status des Ziels. Sie können hier auch ausgewählte Eigenschaften ändern. Auf der Registerkarte Dauerhafte Abonnements finden Sie Informationen über dauerhafte Abonnements, die für diesen Broker erstellt wurden.

Klicken Sie auf die folgenden Links, um nähere Informationen über die Verwendung der Optionen auf diesen Registerkarten zu erhalten.
