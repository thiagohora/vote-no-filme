Überblick

Mit Hilfe der Steuerelemente der Administrationskonsole können Sie mit einem oder mehreren Message Queue-Brokern und Objektspeichern kommunizieren.

Die Administrationskonsole ist in fünf Bereiche unterteilt (siehe Abbildung).

  • 1 - Menüleiste
  • 2 - Symbolleiste
  • 3 - Navigationsbereich
  • 4 - Ergebnisbereich
  • 5 - Statusbereich

Mit Hilfe der Menüs in der Menüleiste oder der Symbolschaltflächen in der Symbolleiste können Sie auf die im Navigationsbereich und im Ergebnisbereich ausgewählten Elemente zugreifen und diese bearbeiten. Im Statusbereich finden Sie Informationen über die zuletzt ausgeführte Aktion.

Der Navigationsbereich enthält eine als Strukturbaum dargestellte Ansicht der Objektspeicher und Broker, mit denen Sie verbunden sind. Objektspeicher enthalten verwaltete Objekte, wie Ziele (Typ Warteschlange oder Thema) oder Verbindungs-Factories. Broker enthalten eine Zusammenstellung von Diensten und physischen Zielen.

  • Wenn Sie sich die Ziele in einem Objektspeicher anzeigen lassen möchten, klicken Sie auf den Ordner Ziele. Im Ergebnisbereich wird daraufhin der Inhalt dieses Ordners angezeigt.
  • Wenn Sie sich Informationen über die Verbindungs-Factories in einem Objektspeicher anzeigen lassen möchten, klicken Sie auf den Ordner Verbindungs-Factories. Im Ergebnisbereich werden daraufhin der Suchname und der Typ der einzelnen Verbindungs-Factories angezeigt.
  • Wenn Sie sich die auf einem Broker verfügbaren Dienste anzeigen lassen möchten, klicken Sie auf den Ordner Dienste. Im Ergebnisbereich wird daraufhin eine Liste der Dienste mit Anschlussnummer und Status angezeigt.
  • Wenn Sie sich Informationen über einen bestimmten Dienst anzeigen lassen oder diese ändern möchten, wählen Sie im Ergebnisbereich den jeweiligen Dienst aus und klicken Sie auf Aktionen > Eigenschaften.
  • Wenn Sie sich Informationen zu den physischen Zielen auf einem Broker anzeigen lassen möchten, klicken Sie auf den Ordner Ziele. Daraufhin wird eine Liste der auf diesem Broker erstellten Ziele angezeigt, der Sie den Typ und, bei Warteschlangen, das Zustellungsverfahren entnehmen können.
  • Wenn Sie sich Informationen über ein bestimmtes Ziel anzeigen lassen oder diese ändern möchten, wählen Sie das jeweilige Ziel im Ergebnisbereich aus und klicken Sie auf Aktionen > Eigenschaften.

Um die Eigenschaften eines Objektspeichers oder eines Brokers ändern zu können, müssen Sie die Verbindung trennen.

Über die grafische Benutzeroberfläche der Administrationskonsole haben Sie Zugriff auf die grundlegenden Funktionen zur Message Queue-Verwaltung. Mit Hilfe des entsprechenden Message Queue-Befehlsprogramms können Sie jedoch auf eine viel größere Zahl von Funktionen zugreifen. Weitere Informationen finden Sie im 6.0 Administrator’s Guide.
