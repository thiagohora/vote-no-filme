Diensteigenschaften

Mit Hilfe dieses Dialogfelds können Sie die Eigenschaften des ausgewählten Message Queue-Dienstes festlegen. Folgende Eigenschaften können geändert werden, ohne dass dadurch der Dienst unterbrochen wird:

Anschlussnummer: Gibt die Nummer des Anschlusses an, den der Portmapper dynamisch dem ausgewählten Dienst zugewiesen hat. Sie können die Option Statisch wählen und einen alternativen (verfügbaren) Anschluss für den Dienst festlegen.

Dienststatus: Gibt den aktuellen Status des Dienstes an.

Aktuelle Zahl zugewiesener Threads: Gibt die Anzahl der Threads an, die diesem Dienst zugewiesen sind.

Aktuelle Zahl Verbindungen: Gibt die Anzahl der Verbindungen an, die diesem Dienst gegenwärtig zugewiesen sind.

Mindestzahl Threads: Gibt an, wie viele Threads aus dem Thread-Pool diesem Dienst minimal zugewiesen sein müssen.

Höchstzahl Threads: Gibt an, wie viele Threads aus dem Thread-Pool diesem Dienst maximal zugeordnet sein dürfen.

Klicken Sie auf OK, um diese Konfigurationsänderungen für den ausgewählten Dienst wirksam werden zu lassen.
