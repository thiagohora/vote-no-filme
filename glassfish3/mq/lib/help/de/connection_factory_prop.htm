Eigenschaften Verbindungs-Factory-Objekt

Mit Hilfe dieses Dialogfelds können Sie die im Dialogfeld Verbindungs-Factory-Objekt hinzufügen festgelegten Verbindungs-Factory-Eigenschaften rekonfigurieren.

Schreibgeschützt: Aktivieren Sie diese Option, wenn das Verbindungs-Factory-Objekt schreibgeschützt sein soll. Bei schreibgeschützten Objekten können Client-Anwendungen die Attribute von Verbindungs-Factory-Objekten nicht ändern.

Durch Rekonfigurieren eines Verbindungs-Factory-Objekts ändern Sie die Eigenschaften, die allen vom Factory-Objekt erzeugten Verbindungen gemein sind. Für das Rekonfigurieren des Verbindungs-Factory-Objekts stehen Ihnen in diesem Dialogfeld eine Reihe von Registerkarten zur Verfügung. Informationen zu den einzelnen Registerkarten erhalten Sie, wenn Sie auf den entsprechenden Link klicken.
