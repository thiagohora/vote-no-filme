3.0 Verbindungs-Handling

Mit Hilfe der Optionen auf der Registerkarte Verbindungs-Handling können Sie grundlegende Informationen zu den Verbindungen rekonfigurieren, die von diesem Factory-Objekt erstellt werden.

Verbindungstyp: Wählen Sie den Verbindungstyp aus.

Broker-Hostname: Geben Sie den Namen des Computers ein, auf dem der Broker gehostet wird, mit dem der Client eine Verbindung herstellen soll.

Broker-Hostanschluss: Geben Sie die Nummer des Portmapper-Anschlusses ein. Der Client verwendet die vom Portmapper angegebenen Informationen, um eine Verbindung mit dem gewünschten Dienst (JMS oder SSLJMS) herzustellen.

Broker-Dienstanschluss: Geben Sie die Nummer des Anschlusses an, der für eine direkte Verbindung zum JMS- oder SSLJMS-Dienst verwendet werden soll. Mithilfe der Option Broker-Hostanschluss können Sie den Portmapper umgehen. Auf diese Weise können Clients mithilfe eines einzigen bekannten Anschlusses Verbindungen über Firewalls hinweg herstellen.

HTTP-URL: Geben Sie den URL ein, über den die Verbindung zum Meldungsdienst erfolgen soll. Der Standardwert lautet http://localhost/imq/tunnel.
