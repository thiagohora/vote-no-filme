Eigenschaften Zielobjekt

Mit Hilfe dieses Dialogfelds können Sie die Eigenschaften von Zielobjekten ändern:

Schreibgeschützt: Aktivieren Sie diese Option, wenn das Objekt schreibgeschützt sein soll. Bei schreibgeschützten Objekten können Client-Anwendungen die Attribute von verwalteten Objekten nicht ändern.

Zielname: Legen Sie in diesem Feld den Namen des physischen Ziels auf dem Broker fest, dem das verwaltete Objekt entspricht. Wenn Sie keinen Namen festlegen, wird der Standardname Untitled_Destination_Object verwendet.

Zielbeschreibung: Geben Sie in diesem Feld eine aussagekräftige Beschreibung für das Zielobjekt ein.

Klicken Sie auf OK, um die vorgenommenen Änderungen der Zielobjekteigenschaften wirksam werden zu lassen.
