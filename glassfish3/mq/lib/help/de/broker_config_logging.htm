Protokollierung

Mit Hilfe der Optionen auf dieser Registerkarte können Sie die Protokollierung für eine Broker-Instanz konfigurieren.

Protokollebene: Gibt die Ebene der Protokollierungsausgabe an, die in einen beliebigen Ausgabekanal geschrieben werden kann. Folgende Kategorien stehen zur Wahl (in absteigender Reihenfolge ihrer Schwere): FEHLER, WARNUNG und INFO. Wählen Sie in der Dropdown-Liste eine der folgenden Kategorien aus: Wenn Sie eine Kategorie auswählen, werden Daten der festgelegten Kategorie sowie aller höheren Kategorien protokolliert. Wenn Sie z. B. INFO wählen, erhalten Sie Daten aus allen drei Kategorien. Wenn Sie dagegen FEHLER wählen, erhalten Sie nur FEHLER-Meldungen. Die Standardeinstellung ist INFO.

Protokoll-Rollover-Größe: Legt fest, ab welcher Protokolldateigröße die Protokollierungsausgabe in eine neue Protokolldatei geschrieben werden soll. Wählen Sie in der Dropdown-Liste eine der folgenden Maßeinheiten aus: Byte, Kilobyte oder Megabyte. Wählen Sie die Option Unbegrenzt aus, um kein Rollover basierend auf der Größe der Protokolldatei festzulegen. Der Standardwert lautet Unbegrenzt.

Protokoll-Rollover-Intervall: Legen Sie fest, nach wie vielen Sekunden die Protokollierungsausgabe in eine neue Protokolldatei geschrieben wird. Wählen Sie die Option Unbegrenzt aus, um kein Rollover basierend auf dem Alter der Protokolldatei festzulegen. Der Standardwert ist 604 800.
