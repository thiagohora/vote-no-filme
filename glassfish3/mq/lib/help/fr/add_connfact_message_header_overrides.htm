Ajouter Objet Classe de connexion > Dérogation d'en-tête de message

Utilisez cet onglet pour déroger aux champs d'en-tête de message JMS spécifiant la persistance, la durée de vie et la priorité des messages. La dérogation à ces valeurs étant susceptible d'interférer avec les utilisateurs actifs ou les concepteurs d'application, veuillez les consulter préalablement à toute modification.

Forçage JMSDeliveryMode Spécifie si le champ JMSDeliveryMode, déterminé par le client, peut faire l'objet d'une dérogation. Par défaut : faux.

Valeur de JMSDeliveryMode Spécifie la valeur de dérogation de JMSDeliveryMode. Les valeurs sont 1 (non-persistant) ou 2 (persistant). Par défaut : 2.

Forçage JMSExpiration Spécifie si le champ JMSExpiration, déterminé par le client, peut faire l'objet d'une dérogation. Par défaut : faux.

Valeur de JMSExpiration Spécifie la valeur de dérogation de JMSExpiration (en millisecondes). Par défaut : 0 (illimité).

Forçage JMSPriority Spécifie si le champ JMSPriority, déterminé par le client, peut faire l'objet d'une dérogation. Par défaut : faux.

Valeur de JMSPriority Spécifie la valeur de dérogation de JMSPriority (un entier entre 0 et 9). Par défaut : 4 (normal).

Forçage des messages dans les destinations temporaires Spécifie si les dérogations s'appliquent aux destinations temporaires. Par défaut : faux.
