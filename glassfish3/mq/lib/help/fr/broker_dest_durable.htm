Abonnements durables

Cet onglet affiche une liste des abonnés durables pour la destination de rubrique active.

Pour chaque abonné, la table spécifie l'identifiant client de l'abonné, le nombre de messages attendant d'être consommés et le statut de l'abonnement durable.

Pour supprimer un abonné durable :

  1. Cliquez sur l'abonné dans la table afin de le sélectionner.
  2. Cliquez sur le bouton Supprimer. Remarquez que la suppression est immédiate ; il est inutile de cliquer sur le bouton OK.

Pour purger les messages d'un abonné :

  1. Cliquez sur l'abonné dans la table afin de le sélectionner.
  2. Cliquez sur le bouton Purger. Remarquez que cette action est immédiate ; il est inutile de cliquer sur le bouton OK.
