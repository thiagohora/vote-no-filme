Propriétés de l'objet de destination

Utilisez cette boîte de dialogue pour modifier les propriétés d'un objet de destination.

Lecture seule Cliquez sur cette case d'option pour modifier le statut lecture seule de l'objet. Si un objet est en lecture seule, les applications client ne pourront pas modifier les attributs de l'objet administré à l'aide d'appels api.

Nom destination Utilisez ce champ pour spécifier le nom de la destination physique sur le courtier à laquelle cet objet administré correspond. Si vous ne spécifiez aucun nom, le nom par défaut est Untitled_Destination_Object.

Description de la destination Utilisez ce champ pour saisir une description utile de cet objet destination.

Cliquez sur le bouton OK pour configurer à nouveau l'objet de destination à l'aide des valeurs spécifiées dans cette boîte de dialogue.
