Propriétés du service

Utilisez cette boîte de dialogue pour spécifier les propriétés du service Message Queue sélectionné. Vous pouvez modifier les propriétés suivantes sans interrompre le service.

Numéro de port Spécifie le numéro du port attribué dynamiquement au service sélectionné par le mappeur de ports. Vous pouvez cocher la case d'option Statique et spécifier un numéro de port (disponible) alternatif pour le service.

Etat du service Spécifie l'état actuel du service.

Nombre actuel de threads alloués Spécifie le nombre de threads attribués à ce service.

Nombre actuel de connexions Spécifie le nombre de connexions attribuées actuellement à ce service.

Nombre min de threads Spécifie le nombre minimal de threads devant être attribués à partir du pool de threads pour ce service.

Nombre max de threads Spécifie le nombre maximal de threads devant être attribués à partir du pool de threads pour ce service.

Cliquez sur OK pour appliquer ces valeurs de configuration au service sélectionné.
