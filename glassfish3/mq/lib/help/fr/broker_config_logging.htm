Journalisation

Utilisez cet onglet pour configurer la journalisation pour une instance courtier.

Niveau du journal Spécifie le niveau de journalisation qui peut être écrit vers tout canal de sortie. Les catégories, par ordre décroissant de gravité, sont ERROR, WARNING et INFO. Utilisez la liste déroulante pour sélectionner une catégorie. Lorsque vous sélectionnez une catégorie, le journal écrit les données de la catégorie spécifiée et des catégories supérieures. Ainsi, si vous sélectionnez INFO, vous obtenez les données des trois catégories. Si vous sélectionnez ERREUR, vous obtenez uniquement les messages ERREUR. La valeur par défaut est INFO.

Taille du report de journal Spécifie la taille du fichier journal déclenchant le report de la sortie sur un nouveau fichier journal. Utilisez la liste déroulante pour sélectionner l'unité : octets, kilo-octets ou méga-octets. Cliquez sur la case d'option Illimité si vous ne voulez pas que la taille du fichier journal déclenche un report. La sélection par défaut est Illimité.

Intervalle de report du journal Spécifie l'âge, en secondes, du fichier journal déclenchant le report de la sortie sur un nouveau fichier journal. Cliquez sur la case d'option Illimité si vous ne voulez pas que l'âge du fichier déclenche un report. La valeur par défaut est 604800.
