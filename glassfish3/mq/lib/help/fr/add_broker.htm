Ajouter courtier

Utilisez cette boîte de dialogue pour créer la représentation d'un courtier dans la fenêtre de navigation de la console d'administration. Vous n'avez pas besoin de lancer le courtier pour cette action. Vous devrez lancer le courtier pour pouvoir vous connecter sur ce dernier.

Étiquette courtier Spécifie le nom que vous souhaitez voir affiché pour ce courtier dans la fenêtre de navigation de la console d'administration. Il n'est pas nécessaire que ce nom soit identique au nom d'instance du courtier.

Hôte Spécifiez le nom de la machine hôte de ce courtier ou conservez la valeur par défaut localhost.

Port principal Spécifie le numéro du port assigné au mappeur de ports. La valeur par défaut est 7676. La combinaison du nom d'hôte et du numéro de port principal identifie le courtier de manière unique. Pour exécuter deux instances d'un courtier sur un même serveur, chaque mappeur de ports du courtier doit avoir un numéro de port différent.

Nom d'utilisateur Spécifie votre nom d'utilisateur. La valeur par défaut pour l'administrateur est admin. Cette information sert à vous authentifier lorsque vous essayez de vous connecter au courtier. La saisie dans cette boîte de dialogue n'est pas sécurisée. Si la sécurité vous semble un élément important, vous devez attendre le moment de connexion où cette information vous sera demandée.

Mot de passe Spécifie votre mot de passe utilisateur. La valeur par défaut pour l'administrateur est admin. Cette information sert à vous authentifier lorsque vous essayez de vous connecter au courtier. La saisie dans cette boîte de dialogue n'est pas sécurisée. Si la sécurité vous semble un élément important, vous devez attendre le moment de connexion où cette information vous sera demandée.

  • Utilisez le bouton OK pour valider toutes les informations saisies et ajouter le courtier.

Pour vous connecter au courtier que vous avez ajouté :

  1. Sélectionnez le courtier dans la fenêtre de navigation de la console d'administration.
  2. Sélectionnez Actions > Se connecter au courtier.

Pour rechercher ou mettre à jour un courtier actif :

  1. Sélectionnez le courtier dans la fenêtre de navigation de la console d'administration.
  2. Connectez-vous s'il n'est pas déjà connecté.
  3. Sélectionnez Actions > Demander/Mettre à jour courtier.

Pour modifier les informations spécifiées dans cette fenêtre pour un courtier particulier, vous devez :

  1. Sélectionner le courtier dans la fenêtre de navigation.
  2. Déconnecter le courtier si ce n'est pas déjà fait. Un courtier n'est pas déconnecté si son icône dans la fenêtre de navigation ressemble à ce qui suit :
    Pour déconnecter un courtier, sélectionnez Actions > Se déconnecter du courtier
  3. Sélectionnez Actions > Propriétés pour ouvrir la fenêtre Propriétés d'un courtier.
