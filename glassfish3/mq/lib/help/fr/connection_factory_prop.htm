Propriétés de l'objet Classe de connexion

Utilisez cette boîte de dialogue pour configurer à nouveau les propriétés de classe de connexion déjà définies dans la fenêtre Ajout de classe de connexion.

Lecture seule Cochez cette case pour que l'objet de classe de connexion soit en lecture seule. Si l'objet est en lecture seule, les applications clientes ne pourront pas modifier les attributs de la classe de connexion à l'aide d'appels api.

En configurant à nouveau l'objet de classe de connexion, vous définissez à nouveau les propriétés communes à toutes les connexions produites par la classe. Vous utilisez les onglets présentés dans la fenêtre de dialogue pour configurer à nouveau la classe de connexion. Cliquez sur l'un des liens suivants pour obtenir des informations sur un onglet particulier.
