Aperçu

Vous utilisez la console d'administration pour communiquer avec un ou plusieurs courtiers Message Queue et magasins d'objets.

La console d'administration est divisée en cinq fenêtres, comme indiqué ci-dessous.

  • 1 - barre de menu
  • 2 - barre d'outils
  • 3 - fenêtre de navigation
  • 4 - fenêtre de résultats
  • 5 - fenêtre de statut

Vous utilisez les menus de la barre de menu ou les icônes de la barre d'outils pour agir sur les éléments sélectionnés dans la fenêtre de navigation et dans la fenêtre de résultats. La fenêtre de statut affiche les informations sur l'action la plus récemment réalisée.

La fenêtre de navigation affiche une arborescence des magasins d'objets et des courtiers auxquels vous êtes connecté. Les magasins d'objets contiennent des objets administrés, pouvant être des destinations (file d'attente ou rubrique) ou des classes de connexion. Les courtiers contiennent un ensemble de services et de destinations physiques.

  • Pour consulter les destinations d'un magasin d'objets, cliquez sur le dossier Destinations. Son contenu s'affiche dans la fenêtre de résultats.
  • Pour consulter des informations sur les classes de connexion d'un magasin d'objets, cliquez sur le dossier Classes de connexion. Le nom de recherche et le type de chaque classe de connexion sont affichés dans la fenêtre de résultats.
  • Pour consulter des informations sur les services disponibles chez un courtier, cliquez sur le dossier Services. Une liste des services, le numéro de port de chaque service et le statut sont affichés dans la fenêtre de résultats.
  • Pour afficher/modifier les informations d'un service particulier, sélectionnez-le dans la fenêtre de résultats et sélectionnez Actions > Propriétés.
  • Pour consulter des informations sur les destinations physiques sur un courtier, cliquez sur le dossier Destinations. Une liste des destinations ayant été créées chez ce courtier s'affiche, avec le type des destinations et, pour une file d'attente, sa politique de livraison.
  • Pour afficher/modifier les informations d'une destination particulière, sélectionnez-la dans la fenêtre de résultats et sélectionnez Actions > Propriétés.

Pour modifier les propriétés d'un magasin d'objets ou d'un courtier, vous devez d'abord vous déconnecter.

La console d'administration vous donne accès aux fonctions de base de l'administration Message Queue via une interface graphique. Cependant, beaucoup d'autres fonctions sont disponibles si vous utilisez l'utilitaire de commande Message Queue adéquat. Pour plus d'informations, consultez le guide Message Queue Administrator's Guide.
