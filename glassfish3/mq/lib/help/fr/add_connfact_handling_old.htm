Gestion des connexions 3.0

Utilisez l'onglet Gestion des connexions pour spécifier les informations essentielles sur les connexions créées par cette classe.

Type de connexion Utilisez ce menu déroulant pour sélectionner le type de connexion.

Nom d'hôte courtier Spécifiez le nom de la machine hébergeant le courtier auquel le client se connecte.

Port hôte courtier Spécifiez le numéro du port mappeur de ports. Le client utilise les informations fournies par le mappeur de ports pour se connecter au service souhaité, à savoir jms ou ssljms.

Port du service Courtier Spécifiez le numéro du port à utiliser pour établir une connexion directe avec le service jms ou ssljms. Utilisez Port du service Courtier pour ignorer le mappeur de ports. Cela permet aux clients de se connecter au travers de pare-feux à l'aide d'un seul port connu.

URL HTTP Spécifiez l'URL qui sera utilisée pour se connecter au service de messages. La valeur par défaut est http://localhost/imq/tunnel.
