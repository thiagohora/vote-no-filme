Ajouter Object Store

Utilisez cette boîte de dialogue pour ajouter un magasin d'objet existant à la fenêtre de navigation de votre console d'administration. Après l'ajout du magasin d'objets, vous pouvez vous y connecter et commencer à lui ajouter des objets de destination et des objets de classe de connexion.

Un magasin d'objets contient les objets que vous créez : objets de destination et objets de classe de connexion. Son contenu est accessible à l'aide d'un code de recherche JNDI standard appelé par l'application client. Un magasin d'objets peut être basé sur des fichiers ou peut être un serveur LDAP. Si vous souhaitez un magasin basé sur des fichiers, vous devez lui créer un dossier. Si vous souhaitez utiliser un serveur LDAP, vous devez le configurer et le lancer.

Étiquette Object Store : Spécifie le nom du magasin d'objets. Ce nom sera affiché dans la fenêtre de navigation de la console d'administration.

Propriétés du service d'affectation des noms JNDI : Utilisez les commandes de cette zone pour définir la liste des propriétés JNDI pour ce magasin d'objets. Utilisez la liste déroulante Nom pour sélectionner un nom de propriété. Après avoir sélectionné le nom de propriété, utilisez le champ Valeur pour saisir la valeur de la propriété.

Voir http://java.sun.com/products/jndi/1.2/javadoc/ pour en savoir plus sur l'interface de programmation d'application (api) JNDI. Vous devez définir les propriétés java.naming.factory.initial et java.naming.provider.url.

Il existe deux valeurs pour la propriété de contexte initial (java.naming.factory.initial) :

  • Magasin de système de fichiers : com.sun.jndi.fscontext.RefFSContextFactory
  • Serveur LDAP : com.sun.jndi.ldap.LdapCtxFactory

Vous devez également spécifier les informations d'authentification. Cependant, si la sécurité vous semble un élément important, vous pouvez attendre le moment de connexion où cette information vous sera demandée

  • Après avoir spécifié les nom et valeur de la propriété, cliquez sur le bouton Ajouter pour l'ajouter à la table.
  • Pour supprimer une propriété que vous avez déjà ajoutée, sélectionnez-la dans la table et cliquez sur le bouton Supprimer.
  • Utilisez le bouton Effacer pour annuler toutes les entrées.
  • Pour configurer à nouveau une propriété, sélectionnez une entrée, modifiez ses valeurs et utilisez le bouton Modifier pour valider la nouvelle configuration de cette propriété.
  • Utilisez le bouton OK pour appliquer tous les paramètres de propriété et ajouter le magasin d'objets.

Pour modifier les propriétés d'un magasin d'objets existant, vous devez suivre la procédure suivante :

  1. Déconnectez-vous du magasin d'objets si ce n'est pas déjà fait. Un magasin d'objets est déconnecté si son icône dans la fenêtre de navigation ressemble à ce qui suit :
    Pour déconnecter un magasin d'objets, sélectionnez-le dans la fenêtre de navigation et sélectionnez Actions > Se déconnecter du magasin d'objets.
  2. Sélectionnez Actions > Propriétés pour afficher la boîte de dialogue Propriétés d'un magasin d'objets.

Pour ajouter un objet de destination au magasin d'objets, cliquez sur le dossier Destinations et sélectionnez Actions > Ajouter objet de destination.

Pour ajouter une classe de connexion au magasin d'objets, cliquez sur le dossier Classe de connexion et sélectionnez Actions > Ajouter Objet Classe de connexion.
