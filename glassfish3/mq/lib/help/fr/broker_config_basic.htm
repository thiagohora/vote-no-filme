De base

Utilisez cet onglet pour spécifier ou parcourir les informations de configuration de base d'une instance courtier.

Version Spécifie la version du courtier.

Nom d'instance Spécifie le nom par défaut imqbroker ou le nom que vous avez assigné au courtier lorsque vous l'avez lancé.

Port principal Spécifie le numéro du port de mappeur de ports du courtier. Le mappeur de ports est responsable de l'attribution dynamique des ports aux services Message Queue. Ce devrait être le même numéro que celui spécifié lorsque vous avez lancé le courtier.

Création automatique de rubriques Spécifie si le courtier est autorisé à créer une destination de type rubrique dès lors qu'un consommateur ou un producteur de message tente d'accéder à une destination inexistante.

Création automatique de files d'attente Spécifie si le courtier est autorisé à créer une destination de type file d'attente dès lors qu'un consommateur ou un producteur de message tente d'accéder à une destination inexistante.

Nombre maximal de consommateurs actifs dans une file d'attente créée automatiquement Spécifie le nombre maximal de consommateurs qui peuvent être actifs dans le cadre d'une livraison avec équilibrage de charge à partir d'une destination de type file d'attente.

Nombre maximal de consommateurs de remplacement dans une file d'attente créée automatiquement : spécifie le nombre maximal de consommateurs pouvant remplacer les consommateurs actifs en cas d'échec de ces derniers pendant une livraison avec équilibrage de charge à partir d'une destination de type file d'attente.
