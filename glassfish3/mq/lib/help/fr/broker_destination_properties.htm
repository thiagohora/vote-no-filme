Propriétés de destination du courtier

Cette boîte de dialogue vous permet de consulter le statut actuel d'une destination courtier et de modifier les propriétés spécifiées pour cette destination.

Utilisez l'onglet De base pour obtenir des informations sur le statut actuel des destinations et modifier les propriétés sélectionnées. Utilisez l'onglet Abonnements durables pour obtenir des informations sur les abonnements durables ayant été créés pour ce courtier.

Le lien suivant offre des informations plus détaillées sur la façon d'utiliser les champs dans ces onglets.
