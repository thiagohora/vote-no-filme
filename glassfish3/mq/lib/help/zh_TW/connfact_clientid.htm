用戶端標識

使用此標籤可重新指定使用者名稱和密碼的預設值，並可重新指定代理程式如何識別用戶端 (ClientID)。ClientID 值主要用於記錄長期用戶。

預設使用者名稱 指定由代理程式來鑑定的預設使用者名稱。此預設值為 guest。

預設密碼 指定由代理程式鑑定的預設密碼。此預設值為 guest。

停用 setClientID() JMS API 指定防止用戶端應用程式使用 API 呼叫變更 ClientID。這樣通常更有利於管理員控制 ClientID 的指定。

如果您核取了此方塊，可用 [配置 ClientID] 欄位指定 ClientID。如果您未指定此值，則系統會指定包含用戶端主機 IP 位址的預設值。這樣做在方便開發者的同時，也阻止了用戶端應用程式建立多個連接。系統不允許兩個連接具有相同的 ClientID。

配置 ClientID 指定用戶端的識別碼。如果您未指定此值，並且已停用了設定 ClientID 的 API 呼叫，則系統會指定包含用戶端主機 IP 位址的預設值。這樣做在方便開發者的同時，也阻止了用戶端應用程式建立多個連接。系統不允許兩個連接具有相同的 ClientID。
