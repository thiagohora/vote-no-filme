物件倉庫屬性

使用此對話方塊新增、刪除或更新已加入到管理主控台導航窗格中的物件倉庫的屬性。若要編輯此對話方塊的欄位，請必須取消物件倉庫與管理主控台的連接，然後再次顯示對話方塊。

JNDI 命名服務屬性：使用此區域中的控制項，為該物件倉庫定義 JNDI 屬性清單。使用 [名稱] 下拉式清單選取屬性名稱。選取屬性名稱之後，使用 [值] 欄位輸入屬性值。

請參閱 http://java.sun.com/products/jndi/1.2/javadoc/，以取得有關 JNDI 應用程式設計介面的資訊。這需要您設定 java.naming.factory.initial 和 java.naming.provider.url 屬性。

初始環境屬性 (java.naming.factory.initial) 可以有兩種值：

  • 檔案系統倉庫：com.sun.jndi.fscontext.RefFSContextFactory
  • LDAP 伺服器：com.sun.jndi.ldap.LdapCtxFactory

而且還需要您指定認證資訊。但如果您擔心安全問題，則可等到連接時由系統提示您指定此類資訊。

  • 指定屬性名稱和值之後，請按一下 [加入] 按鈕，將屬性加入表中。
  • 若要刪除現有的屬性，請選取 [刪除] 按鈕將其刪除。
  • 使用 [清除] 按鈕清除全部項目。
  • 選取一個項目，變更它的值，然後用 [變更] 按鈕重新配置此屬性。
  • 使用 [確定] 按鈕確定所有屬性設定。

若要在重新配置後重新連接到物件倉庫：請在導航窗格中選取物件倉庫並選擇 [動作] > [連接至物件倉庫]。
