3.0 連接處理

使用 [連接處理] 標籤指定將由此 Factory 建立的連接的基本資訊。

連接類型 使用此下拉式清單選取連接類型。

代理程式主機名稱 指定用戶端將連接至的代理程式之主機名稱。

代理程式主機連接埠 指定連接埠對映器的連接埠的編號。用戶端使用連接埠對映器所提供的資訊，連接理想的服務：jms 或 ssljms。

代理程式服務連接埠 指定要使用的連接埠數，以直接連接 jms 或 ssljms 服務。使用 [代理程式服務連接埠] 以略過連接埠對映器。如此可讓用戶端使用單一已知的連接埠，透過防火牆進行連接。

HTTP URL 指定用於連接至訊息服務的 URL。依預設，此為 http://localhost/imq/tunnel。
