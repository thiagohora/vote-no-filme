基本資訊

使用此標籤指定或瀏覽代理程式實例的基本配置資訊。

版本 指定代理程式的版本。

實例名稱 指定預設名稱 imqbroker，或您在啟動代理程式時所指定的名稱。

主連接埠 指定代理程式的連接埠對映器連接埠的編號。連接埠對映器負責動態地為 Message Queue 服務指定連接埠。這應與您啟動代理程式時指定的連接埠編號相同。

自動建立主題 當訊息使用者或訊息產生者試圖存取一個不存在的目標時，指定是否允許代理程式建立主題目標。

自動建立佇列 當訊息使用者或訊息產生者試圖存取一個不存在的目標時，指定是否允許代理程式建立佇列目標。

自動建立最大作用中使用者數佇列 指定最大使用者數，此數值可作用於來自佇列目標的負載平衡發送。

自動建立最大備份使用者數佇列 指定最大備份使用者數，如果作用中的使用者無法從佇列目標進行負載平衡發送，則這些使用者可以取代作用中的使用者。
