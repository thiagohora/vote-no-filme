訊息傳送

使用此標籤配置此代理程式實例的訊息傳送能力和訊息容量。

系統中訊息的最大數目 指定記憶體和磁碟中的最大訊息數目。其他訊息將被拒絕。預設值為 [無限制]。

系統中最大訊息容量 指定記憶體和磁碟中 (因為有資料交換) 的訊息最大總容量 (以 Byte、Kbyte 或 MB 為單位)。其他訊息將被拒絕。預設值為 [無限制]。使用下拉式清單指定計算容量的單位：Byte、Kbyte 或 MB。

最大訊息容量 指定訊息內文的允許最大容量 (以 Byte、Kbyte 或 MB 為單位)。任何大於該設定的訊息將被拒絕。預設值為 70 MB。使用下拉式清單指定計算容量的單位：Byte、Kbyte 或 MB。
