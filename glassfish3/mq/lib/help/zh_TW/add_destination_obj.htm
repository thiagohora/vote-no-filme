新增目標物件

使用此對話方塊，將目標物件加入物件倉庫。

目標受管理物件 是代理程式中與供應商無關的實體目標 (佇列或主題) 的表示法。

查找名稱 使用此欄位指定受管理物件的 JNDI 查找名稱。此名稱可以與目標名稱相同；唯一的限制是在物件倉庫的上下文中，該名稱必須是唯一的。

目標類型 按一下目標類型所對應的單選按鈕：佇列或主題。

唯讀 按一下此核取方塊，使物件為唯讀。如果物件為唯讀，則用戶端應用程式將無法透過 API 呼叫修改受管理物件的屬性。

目標名稱 使用此欄位指定受管理物件所對應的代理程式上的實體目標的名稱。如果您未指定名稱，則使用預設名稱 Untitled_Destination_Object。

目標描述 使用此欄位輸入該目標物件的有用描述。

  • 按一下 [重設為預設值] 按鈕，將全部欄位重設為其預設值。
  • 按一下 [確定]，建立目標物件。

若要變更目標物件的屬性：

  1. 在管理主控台的導航窗格中選取 Destinations 資料夾。
  2. 在管理主控台的結果窗格中選取目標物件。
  3. 選擇 [動作] > [屬性]，顯示 [目標物件屬性] 對話方塊。
